# 엔딩 (Endings)

> *"시간에게 묻는다. 흐름과 정지, 어느 것이 구원인가?"*

---

## 엔딩 개요

| 엔딩 | 조건 | 분위기 |
|------|------|--------|
| **배드 엔딩 A** | 시계를 떨어뜨림 | 비극적 순환 |
| **배드 엔딩 B** | 아무것도 안 함 | 영원한 정지 |
| **트루 엔딩** | 기어 파괴 | 희망적 해방 |
| **히든 엔딩** | 특수 조건 | 열린 결말 |

---

## 배드 엔딩 A: 순환의 고리 (The Closed Loop)

### 조건
4장에서 "회중시계를 떨어뜨린다" 선택

### 시나리오

#### 장면 1: 떨어지는 시계
```
(늙은 크론의 손에서 시계가 미끄러짐)
(슬로우 모션으로 공중에 떠감)
(어린 크론의 눈앞에 멈춤)
```

#### 장면 2: 문이 닫힘
```
어린 크론: "할아버지..."
(어린 크론이 시계를 줍는다)
(천천히 문을 닫는다)
(닫힌 문 너머로 째깍 소리)
```

#### 장면 3: 루프
```
[화면 페이드 아웃]
[프롤로그의 첫 장면으로 전환]
[다시 어린 크론이 다락방에 있음]
```

### 엔딩 내레이션
> *"시간을 다루는 힘, 그것은 다시 작은 시계 가게의 다락방에서"*
> *"비극을 잉태한 채 째깍이기 시작했다."*

### 엔딩 크레딧 후
```
[어린 크론이 장난감 총을 내려놓으려는 장면]
[총성]
[루프 반복 암시]
```

---

## 배드 엔딩 B: 영원의 순간 (The Eternal Moment)

### 조건
4장에서 "아무것도 하지 않는다" 선택

### 시나리오

#### 장면 1: 모라의 승리
```
모라: "...고마워, 크론."
모라: "네가 아무것도 안 해줘서."
(모라가 마스터 기어를 완전히 파괴)
```

#### 장면 2: 세계의 정지
```
(모든 것이 멈춤)
(색채가 사라지고 회색빛으로)
(시간이 흐르지 않음)
```

#### 장면 3: 박제된 세계
```
[정지된 마을 풍경]
[웃고 있는 아이들 - 영원히 그 표정]
[떨어지다 멈춘 나뭇잎]
[모라만이 움직임]
```

### 모라의 독백
```
모라: "이제 아무도 죽지 않아."
모라: "아무도 늙지 않아."
모라: "가장 아름다운 순간에... 영원히."
(모라의 얼굴에 외로움)
모라: "...외롭네."
```

### 엔딩 내레이션
> *"흐르지 않는 시간 속에서,"*
> *"그녀는 홀로 영원을 걸었다."*
> *"구원인가, 감옥인가. 그 답은 아무도 모른다."*

---

## 트루 엔딩: 새로운 시간 (The New Time)

### 조건
4장에서 "마스터 기어를 파괴한다" 선택

### 시나리오

#### 장면 1: 기어 파괴
```
늙은 크론: "미안해, 모라."
늙은 크론: "네 방법은... 답이 아니야."
(크론이 마지막 힘으로 기어를 부숨)
(찌그러지는 금속 소리)
```

#### 장면 2: 다락방의 소멸
```
모라: "안 돼! 뭐하는 거야!"
(다락방이 무너지기 시작)
(공허가 아닌, 빛으로 가득 참)
(모든 기어가 흩어짐)
```

#### 장면 3: 늙은 크론의 소멸
```
늙은 크론: (미소) "이제... 자유야."
늙은 크론: "더 이상 루프는 없어."
(늙은 크론의 몸이 빛의 입자로 흩어짐)
(회중시계도 함께 사라짐)
```

#### 장면 4: 어린 크론
```
[시계 가게 - 평범한 아침]
어린 크론: (잠에서 깸) "...꿈?"
어린 크론: "이상한 꿈이었어."
```

#### 장면 5: 친구들과의 재회
```
[학교 운동장]
모라: "크론! 늦었어!"
레이: "오늘도 경찰 놀이할 거지?"
벨로: "내 작은 발명품 보여줄까?"
어린 크론: (미소) "응!"
```

#### 장면 6: 열린 미래
```
[네 친구가 함께 걸어가는 뒷모습]
[태양이 지평선 위로 떠오름]
[시계 가게는 평범한 골동품 가게]
[다락방은... 그냥 다락방]
```

### 엔딩 내레이션
> *"시계 가게도, 다락방도, 회중시계도 사라졌다."*
> *"하지만 시간은 멈추지 않고 미래로 흘렀다."*
> *"알 수 없는 미래. 정해지지 않은 운명."*
> *"그것이야말로, 진정한 자유였다."*

### 엔딩 크레딧 후
```
[성인이 된 네 친구의 실루엣]
[각자 다른 모습이지만, 함께 웃고 있음]
[자막: "시간은 흐른다. 그리고 우리는 변한다."]
```

## 히든 엔딩: 시계공의 유산 (The Clockmaker's Legacy)

### 🆕 해금 조건 (상세)

> [!IMPORTANT]
> 히든 엔딩은 **모든 조건을 충족**해야 진입 가능합니다.
> 하나라도 미충족 시 일반 분기점으로 진행됩니다.

| 조건 | 상세 요구 사항 | 확인 방법 |
|------|---------------|-----------|
| **단서 완전 수집** | 100개 단서 중 100개 | 추리 보드 100% |
| **추론 완전 완료** | 41개 추론 중 41개 | 추론 탭 100% |
| **노화 제한** | 노화 게이지 50% 이하 유지 | UI 게이지 확인 |
| **특정 대화 선택** | 4장에서 모라에게 "이해한다" 선택 | 대화 분기 |
| **조연 전원 대화** | 티카, 오리, 그레이 모두와 대화 | NPC 체크리스트 |
| **숨겨진 기어 수집** | 3개 숨겨진 기어 파편 | 인벤토리 확인 |

### 숨겨진 기어 위치

| 기어 | 위치 | 획득 방법 |
|------|------|-----------|
| **설계자의 파편** | 프롤로그 - 다락방 거울 뒤 | 리와인드로 거울 복원 후 조사 |
| **영원의 파편** | 3장 던전 - 보스 룸 숨겨진 방 | 클론 3개로 발판 동시 밟기 |
| **시간의 파편** | 4장 - 모라 대결 전 다락방 바닥 | 노화 50% 이하일 때만 출현 |

### 해금 확인 연출

조건 충족 시, 4장 분기점에서 추가 선택지가 나타납니다.

```
[4] "이 루프의 근원에 대해 묻는다" ← 히든 선택지

(선택 시)
늙은 크론: "잠깐, 모라. 한 가지 묻고 싶어."
늙은 크론: "이 시계... 이 다락방... 대체 누가 만들었어?"
모라: "...?"
모라: "네가 그걸 물을 줄 몰랐어."
모라: "알겠어. 그렇다면..."

(시간이 멈추고 새로운 존재가 나타남)
```

### 시나리오


#### 특수 장면: 수수께끼의 존재
```
(기어를 파괴하려는 순간, 시간이 멈춤)
(다락방에 또 다른 존재가 나타남)
(로브를 입은 노인 - 정체불명)
```

#### 대화
```
시계공: "너는... 나를 이해했구나."
크론: "당신은 누구야?"
시계공: "처음이자 마지막. 시간의 설계자."
시계공: "이 루프를 만든 것도, 끊으려 한 것도 나야."

크론: "왜 이런 일을?"
시계공: "시간은 흘러야 해. 하지만 지켜볼 존재도 필요해."
시계공: "너는 그 자격을 증명했어."
시계공: "자유의지로 흐름을 선택했으니."
```

#### 선택
```
시계공: "선택해라. 평범한 삶으로 돌아갈 것인가,"
시계공: "아니면 새로운 시계공이 될 것인가."

[1] 돌아간다 → 트루 엔딩과 동일
[2] 시계공이 된다 → 히든 엔딩 전용
```

#### 시계공 선택 시
```
(크론이 시계공의 로브를 받음)
(새로운 회중시계가 생성됨)
(다락방이 재건됨)

시계공: "이제 네가 지켜라. 시간의 흐름을."
시계공: "그리고 언젠가, 다음 후계자를 찾아라."
```

### 엔딩 내레이션
> *"시간을 다루는 힘, 그것은 새로운 손에 넘겨졌다."*
> *"다락방은 다시 째깍이기 시작했다."*
> *"하지만 이번에는, 자유의지로."*

### 엔딩 크레딧 후
```
[새로운 다락방]
[어른이 된 크론이 시계를 손질하고 있음]
[문을 두드리는 소리]
[어린아이의 목소리: "여기가 시계 가게예요?"]
[크론: (미소) "...들어오렴."]
```

## 핵심 테마 정리

### 각 엔딩이 전하는 메시지

| 엔딩 | 메시지 |
|------|--------|
| **배드 A** | 변화를 거부하면 같은 비극이 반복된다 |
| **배드 B** | 고통을 피하려 모든 것을 멈추면 의미도 사라진다 |
| **트루** | 불확실한 미래라도 흐르는 것이 진정한 삶이다 |
| **히든** | 책임을 지고 지키는 것도 하나의 선택이다 |

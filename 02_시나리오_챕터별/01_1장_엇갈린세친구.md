# 1장: 엇갈린 세 친구 (Three Diverged Friends)

> *"15년이라는 시간은 친구를 낯선 사람으로 만들기에 충분했다."*

---

## 개요

| 항목 | 내용 |
|------|------|
| **플레이 시간** | 약 2-3시간 |
| **시간대** | Year 25 (사건 후 15년) |
| **목적** | 용의자 탐색, 친구들의 변화 확인, 모라와 동맹 |
| **분위기** | 낯섦, 추리, 긴장 |

---

## 1-1: 미래 도시 도착

### 상황
다락방 문을 열고 나온 크론. 눈앞에 펼쳐진 것은 15년 후의 세계.

### 환경 묘사
```
┌──────────────────────────────────────────────────┐
│                Neo-Chronopolis                    │
│                                                  │
│   ▓▓▓▓▓     네온 간판    ▓▓▓▓▓                 │
│   ▓  ▓▓▓▓▓     ☆       ▓▓▓  ▓                 │
│   ▓▓▓  ▓    증기 파이프   ▓▓▓▓▓                 │
│      ║          ║                               │
│   ═══╬══════════╬═══  녹슨 철로                 │
│      ║   ○      ║       ← 크론                  │
│   ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓                            │
└──────────────────────────────────────────────────┘
```

### 크론의 반응
```
크론: "여기가... 우리 마을이야? 이게 대체..."
크론: "15년? 정말 15년이 지난 거야?"
```

### 첫 번째 정보 수집
- NPC와 대화: "시계 가게? 아, 그 집. 15년 전에 화재가 났다던데..."
- 게시판: 레이 형사의 기사, 벨로 박사의 연구 발표, 모라 검사의 판결 뉴스

---

## 1-2: 레이 탐색 - 치안국

### 목적
첫 번째 용의자 레이를 만납니다. 총소리 → 형사 → 권총 → 의심.

### 치안국 묘사
- 어두운 조명, 축축한 공기
- 부패의 냄새 (비유적으로)
- 경찰들이 뇌물을 주고받는 장면 목격

### 레이 첫 등장
```
레이: "누구야? 이 시간에 치안국을 찾아오는 놈이..."
레이: "...잠깐. 그 얼굴..."
크론: "레이? 레이 맞지? 나야, 크론!"
레이: "...크론? 15년 동안 어디 있었어? 넌 죽은 줄..."
```

### 정보 획득
| 질문 | 레이의 답변 |
|------|-------------|
| 할아버지에 대해 | "화재로 죽었다고 들었어. 시체는 못 찾았지만." |
| 그날 밤 총소리 | "...총? 그때 나는 학교에 있었어." (눈을 피함) |
| 지금 하는 일 | "형사야. 뭐, 돈 되는 일도 좀 하고." |

### 플레이어의 판단 유도
- 레이는 권총을 차고 있음
- 눈을 피하며 대답함
- "용의자 1"로 등록

---

## 1-3: 벨로 탐색 - 연구소

### 목적
두 번째 용의자 벨로를 만납니다. 시간 기술 → 동기 의심.

### 연구소 묘사
- 위험 표시가 곳곳에
- 시간이 왜곡된 구역 (느리게/빠르게 흐르는 구역)
- 연구원들이 기피하는 "금지 구역"

### 벨로 첫 등장
```
벨로: "누구야? 여긴 허가된 인원만 들어올 수 있어!"
벨로: "...크론? 크론이라고? 하, 이건 흥미롭군."
벨로: "15년 동안 어디 있었어? 내 연구에 도움이 될 것 같은데..."
```

### 중요 힌트 제공
```
벨로: "시간을 직접 제어하는 연구를 하고 있어."
벨로: "근데 이상하지 않아? 15년 전 그날, 시간에 뭔가 이상이 있었어."
벨로: "마치 누군가 시간을 '편집'한 것처럼..."
```

### 플레이어의 판단 유도
- 벨로는 시간 기술을 연구 중
- 동기가 있어 보임
- "용의자 2"로 등록

---

## 1-4: 모라와의 재회 - 법원 타워

### 목적
모라를 만나 동맹을 맺습니다. (진짜 최종 보스의 복선)

### 법원 타워 묘사
- 최상층까지 올라가는 엘리베이터
- 정의를 상징하는 천칭 조각상
- 모라의 집무실: 깔끔하고 차가움

### 모라 첫 등장
```
모라: "크론..."
모라: "15년이야. 어디 있었어?"
크론: "다락방에서... 시간이 이상하게 흘러서..."
모라: "...다락방. 시간. 그래, 네 말을 믿어."
```

### 동맹 제안
```
모라: "할아버지의 진짜 죽음 원인을 찾고 싶구나?"
모라: "나도 궁금했어. 그날 밤, 분명 뭔가 이상했거든."
모라: "같이 찾아보자. 나는 법적 권한이 있으니까."
```

### 복선 대사
```
모라: "참, 크론. 한 가지 물어볼게."
모라: "시간이 계속 흐르는 게... 정말 좋은 일이라고 생각해?"
크론: "...무슨 뜻이야?"
모라: (미소) "아무것도 아니야. 그냥 철학적인 질문이야."
```

## 1-5: 단서 정리 - 모라의 도움

### 상황
모라가 수사 권한을 사용해 정보를 제공합니다.

### 정리된 정보

| 용의자 | 동기 | 증거 | 알리바이 |
|--------|------|------|----------|
| **레이** | 할아버지와 다툼? | 권총 소지 | 학교에 있었다고 주장 |
| **벨로** | 시간 기술 탐욕 | 연구 중 | 실험실에 있었다고 주장 |
| **모라** | ??? | 없음 | 집에 있었음 (확인 불가) |

### 모라의 제안
```
모라: "알리바이를 확인하려면 직접 과거로 가봐야 해."
모라: "네가 그 능력이 있잖아. 다락방을 통해서."
크론: "과거로? 어떻게?"
모라: "작은 톱니바퀴를 끼우면 시간이 빨라지고..."
모라: "큰 톱니바퀴를 끼우면 시간이 느려진다고 했잖아."
모라: "그럼 역으로 조작하면 과거로 갈 수 있지 않을까?"
```

## 1-6: 챕터 보스 - 치안국의 그림자

### 상황
치안국을 나서려는 크론을 부패 경찰들이 막습니다.

### 보스: 틱톡 경비병 x3 + 중간 보스 "시간 먹는 자"

#### 전투 흐름
1. 틱톡 경비병 처리 (튜토리얼 복습)
2. "시간 먹는 자" 등장 - 레이가 풀어놓은 괴물
3. 레이는 멀리서 지켜보다가 사라짐

#### 보스 패턴
| 패턴 | 설명 | 대응 |
|------|------|------|
| 시간 흡수 | 플레이어 주변 시간 느려짐 | 크로노 시프트로 탈출 |
| 시간 폭발 | 축적된 시간을 폭발시킴 | 타임 앵커로 막음 |
| 분열 | 3초 전의 자신 소환 | 본체 집중 공격 |

### 전투 후
```
크론: "레이... 네가 이런 짓을?"
(레이는 이미 없음)
크론: "도망갔어..."
```

## 1-7: 챕터 엔딩 - 의심과 동맹

### 모라와의 대화
```
모라: "레이가 용의자인 것 같아?"
크론: "모르겠어. 분명 뭔가 숨기고 있어."
모라: "벨로도 수상해. 시간 기술을 연구하고 있잖아."
모라: "과거로 가서 확인해봐. 나는 여기서 조사를 계속할게."
```

### 다락방 귀환
- 크론이 다락방으로 돌아옴
- 다음 챕터를 위한 기어 슬롯 선택지 등장

### 선택지 (가짜)
```
[레이의 과거를 확인한다]
[벨로의 과거를 확인한다]
```

> [!NOTE]
> 어느 쪽을 선택해도 2장에서 둘 다 확인하게 됩니다.

## 챕터 요약

### 획득 아이템
- 레이의 낡은 배지 (유물)
- 벨로의 연구 노트 조각
- 모라의 연락 장치

### 해금 요소
- 과거로 이동하는 기어 조합 힌트
- 기어 사이의 세계 진입 가능

### 스토리 진행
- 용의자 2명 확보 (레이, 벨로)
- 조력자 1명 확보 (모라)
- 할아버지의 죽음이 단순하지 않음을 확인

# 4장: 최후의 심판 (The Final Judgment)

> *"운명의 그날, 다락방 문 앞에 선 것은 다른 누구도 아닌 모라였다."*

---

| 항목 | 내용 |

## 4-1: "그날"의 다락방

### 상황
늙은 크론이 "그날"의 다락방에 도착합니다.

### 환경 묘사
- 익숙한 다락방
- 하지만 모든 것이 새롭게 보임
- 창밖은 아직 평화로운 마을

### 독백
```
늙은 크론: "여기야. 모든 게 시작되고... 끝나는 곳."
늙은 크론: "아직 일어나지 않았어. 총소리가 울리기 전."
늙은 크론: "범인을 기다리면 돼."
```

---

## 4-2: 대기

### 상황
크론이 다락방에서 범인을 기다립니다.

### 시간 경과 연출
```
[오후 3시] - 아무 일 없음
[오후 5시] - 저녁이 됨
[오후 7시] - 어둠이 내림
```

### 긴장감 조성
- 멀리서 발소리
- 계단을 올라오는 소리
- 문 앞에서 멈추는 소리

---

## 4-3: 범인의 등장

### 상황
문이 열리고, 그 너머에 **모라**가 서 있습니다.

### 연출
```
(문이 천천히 열림)
(익숙한 얼굴)
(손에는 권총)

모라: "...알고 있었구나, 크론."
늙은 크론: "모라."
모라: "아니, 이제는 '할아버지'라고 불러야 하나?"
```

### 충격
```
늙은 크론: "네가... 범인이었어?"
모라: "범인? 아니. 나는 구세주야."
늙은 크론: "무슨 소리야?"
```

---

## 4-4: 모라의 진실

### 상황
모라가 자신의 동기를 설명합니다.

### 대사 (핵심)
```
모라: "난 봤어, 크론."
모라: "네가 시간을 남용할수록 세계가 어떻게 되는지."
모라: "인과율이 무너지고, 공허가 침식해오고..."
모라: "결국 모든 것이 소멸해."

늙은 크론: "그건... 내 잘못이야?"
모라: "잘못? 아니, 필연이야."
모라: "시간이 흐르는 한, 파멸은 필연적이야."
모라: "그래서 멈춰야 해."
```

### 🆕 "인과의 비명" 진실 공개

모라가 총성의 진실을 밝힙니다.

```
늙은 크론: "그날 밤 총소리... 네가 쐈어?"
모라: "총? 아니."
모라: "그건 총소리가 아니야, 크론."

(모라가 데이터 패드를 보여줌)

모라: "그레이가 분석한 결과야."
모라: "화약 폭발음이 아니야. 시간 입자 붕괴음."
모라: "네가 수십 년간 개입한 시간이 한 점에 모였어."
모라: "그 압력이 터진 거야. 인과의 비명."

늙은 크론: "그럼... 총알은?"
모라: "없었어. 처음부터."
모라: "티카가 처음 발견했지. 총알이 없다고."

(플래시백 - 티카의 증언)
티카: "총소리는 분명히 들었어요. 근데... 총알이 없었어요."

(플래시백 - 오리의 증언)
오리: "총상이 아니야. 안에서 바깥으로 터진 것 같았어."

(플래시백 - 그레이의 분석)
그레이: "이 음파 패턴은 화약 폭발이 아니야."

모라: "알겠어, 크론?"
모라: "할아버지를 죽인 건... 누구도 아니야."
모라: "시간을 개입한 네 자신이야."

늙은 크론: "...내가? 내가 할아버지를... 나를 죽인 거야?"
```

> [!CAUTION]
> **핵심 반전**: 범인을 찾아 헤맨 모든 추리가 무의미했습니다.
> 진짜 "범인"은 존재하지 않았고, 원인은 시간 개입 그 자체였습니다.

### 모라의 목적
```
모라: "이 다락방의 마스터 기어를 파괴하면,"
모라: "시간은 영원히 멈춰."
모라: "더 이상 아무도 늙지 않아."
모라: "더 이상 아무도 죽지 않아."
모라: "가장 아름다운 순간에... 박제되는 거야."
```

### 🆕 모라의 실패한 시도들

```
늙은 크론: "왜 이런 극단적인 방법을?"
모라: "극단적? 난 이미 다 시도해봤어."
모라: "너를 죽이려 했어. 루프로 부활했지."
모라: "시계를 부쉈어. 15분 후에 재생했어."
모라: "다락방을 봉인했어. 다른 문이 나타났어."
모라: "법을 만들었어. 시간 범죄는 증거가 안 남아."
모라: "아무것도 안 됐어, 크론."
모라: "유일한 방법은... 시간 자체를 멈추는 거야."
```

## 4-5: 철학적 대립

### 늙은 크론의 반박
```
늙은 크론: "그게 진짜 구원이야?"
늙은 크론: "시간이 멈추면... 성장도 없어."
늙은 크론: "만남도, 이별도, 변화도 없어."
늙은 크론: "그건 삶이 아니야."

모라: "삶? 흐르는 건 결국 썩어 문드러져."
모라: "나는 세계를 가장 아름다운 순간에 보존할 거야."
```

### 대립 구도
| 크론 | 모라 |
|------|------|
| 시간은 흘러야 한다 | 시간은 멈춰야 한다 |
| 변화가 있어야 의미 | 변화가 있어서 파멸 |
| 순리의 수호자 | 통제의 집행자 |

## 4-6: 최종 보스전 - 모라

### 보스: 심판자 모라 (Mora, The Judge)

#### 페이즈 1: 권총 사격
| 패턴 | 설명 | 대응 |
|------|------|------|
| 정조준 | 빨간 레이저 후 발사 | 크로노 시프트 회피 |
| 난사 | 부채꼴 사격 | 타임 앵커로 느리게 |
| 리로드 | 탄창 교체 중 틈 | 공격 기회 |

#### 페이즈 2: 시간 조작
모라가 **제4의 기어(정지 기어)**를 활성화합니다.

| 패턴 | 설명 | 대응 |
|------|------|------|
| 시간 정지 | 화면 일부 정지 | 정지 구역 피하기 |
| 순간이동 | 위치 교체 | 에코 클론 활용 |
| 연속 정지 | 플레이어만 정지 | 리와인드로 탈출 |

#### 페이즈 3: 공허 소환
마스터 기어에 총격을 가하며 공허가 침식해옵니다.

| 패턴 | 설명 | 대응 |
|------|------|------|
| 공허 확산 | 바닥이 보라색으로 | 안전 지대로 이동 |
| 중력 상실 | 부유 상태 | 조작 적응 필요 |
| 최후의 일격 | 마스터 기어 파괴 시도 | 저지해야 함 |

## 4-7: (분기점) 그 날의 진실

### 상황
전투 중, "그 순간"이 다가옵니다.

### 연출
```
모라: "이제 끝이야, 크론!"
(모라가 마스터 기어에 방아쇠를 당김)

탕!

(총성 - 하지만 화약 소리가 아님)
(시간의 축이 부서지는 비명)
```

### 다락방의 붕괴
```
- 중력이 사라짐
- 늙은 크론이 공중으로 떠오름
- 세계가 보랏빛 공허로 물듦
- 문이 열림
```

### 과거와의 조우
```
(문밖에 어린 크론이 서 있음)
(프롤로그의 그 장면)
(겁에 질린 눈으로 올려다보는 과거의 나)
```

## 4-8: 선택의 순간

### 분기점
플레이어에게 최종 선택이 주어집니다.

### 선택지
```
[1] 회중시계를 떨어뜨린다 (루프 유지)
[2] 마스터 기어를 완전히 파괴한다 (루프 종료)
[3] 아무것도 하지 않는다 (모라의 승리)
```

---

## 분기별 전개

### 선택 1: 회중시계를 떨어뜨린다

#### 연출
```
늙은 크론: (소리치려 함) "문을 닫아! 나오지 마!"
(하지만 목소리가 나오지 않음)
(의식이 흐려짐)
(손에서 회중시계가 빠져나감)
(시계가 공중에 떠서 어린 크론 앞에 머무름)
```

#### 어린 크론
```
(어린 크론이 시계를 줍는다)
(문을 닫는다)
```

→ **배드 엔딩으로 연결**

### 선택 2: 마스터 기어를 파괴한다

#### 연출
```
늙은 크론: "미안해, 모라. 하지만 시간은 흘러야 해."
(마지막 힘을 짜내 기어에 손을 뻗음)
(기어를 부숨)
```

→ **진 엔딩으로 연결**

### 선택 3: 아무것도 하지 않는다

#### 연출
```
(모라가 승리함)
(시간이 완전히 정지)
(세계가 얼어붙음)
```

→ **또 다른 배드 엔딩으로 연결**

## 챕터 요약

### 핵심 공개
- 범인 = 모라
- 모라의 동기 = 파멸을 막기 위한 시간 정지
- 총소리의 정체 = 마스터 기어 파괴음

### 철학적 테마
- 흐름 vs 정지
- 변화 vs 보존
- 자유의지 vs 운명

### 게임플레이
- 최종 보스 3페이즈
- 공허 환경에서의 전투
- 선택에 따른 엔딩 분기

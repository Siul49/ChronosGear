# 프롤로그: 0의 시간 (The Zero Hour)

> *"시간을 다루는 힘, 그것은 작은 시계 가게의 다락방에서 째깍이고 있었다."*

---

## 개요

| 항목 | 내용 |
|------|------|
| **플레이 시간** | 약 20-30분 |
| **목적** | 세계관 소개, 기본 조작 학습, 사건의 시작 |
| **분위기** | 평화 → 공포 → 혼란 → 결의 |

---

## 씬 구성

### 씬 1: 평온한 오후

#### 상황
어린 크론(10세)이 학교에서 돌아옵니다. 문을 열고 시계 가게를 지나 다락방으로 올라갑니다.

#### 연출
- 1층: 수백 개의 시계가 째깍거림 (ASMR 같은 배경음)
- 2층: 할아버지가 "학교 잘 다녀왔니?"라고 물음
- 다락방: 크론의 방, 장난감 총과 경찰 배지가 있음

#### 대사
```
할아버지: "크론, 학교 잘 다녀왔니?"
크론: "네! 오늘 모라가 범인 역할 했어요. 제가 잡았죠!"
할아버지: "허허, 그래. 그런데 크론아..."
크론: "네?"
할아버지: "다락방 문 밖은 절대 함부로 열면 안 된다. 알겠지?"
크론: "...알겠어요."
```

> [!NOTE]
> 이 대사는 후반부에 반복됩니다. 늙은 크론이 어린 자신에게 하려던 말이기도 합니다.

---

### 씬 2: 경찰 놀이의 여운

#### 상황
다락방에서 혼자 경찰 놀이를 하는 크론. 장난감 총을 들고 상상 속 범인을 쫓습니다.

#### 튜토리얼 요소
- **기본 이동**: WASD / 좌측 스틱
- **상호작용**: E / A버튼 (물건 조사)
- 조사 가능 오브젝트: 장난감 총, 할아버지 사진, 오래된 시계들

#### 🆕 첫 번째 단서 수집
| 조사 대상 | 발견 내용 | 추리 보드 |
|-----------|-----------|-----------|
| 할아버지 사진 | "할아버지와 어린 크론의 사진. 행복해 보인다." | 📦 단서 획득 |
| 책상 위 메모 | "다락방 문틀의 세 슬롯을 기억할 것" | 📜 기록 획득 |
| 오래된 시계 | "멈춰 있다. 왜 여기만 멈춰 있을까?" | 👁️ 환경 단서 |

#### 시간 경과
플레이어가 방을 돌아다니는 동안 창밖이 점점 어두워집니다 (저녁 → 밤).

---

### 씬 3: 총성

#### 상황
약 1시간(게임 내 시간)이 지납니다. 크론이 장난감 총을 책상에 내려놓으려는 순간—

#### 연출
```
[화면 전체가 잠시 흔들림]

효과음: 탕!

[크론이 움찔함, 장난감 총을 다시 꽉 쥠]
```

#### 내레이션
> *"실제로는 처음 들어보는 총 소리였다. 생각보다 커서, 그리고 가까워서 식은 땀을 흘리며 문에 다가섰다."*

---

### 씬 4: 문 앞에서의 고민

#### 상황
크론은 다락방 문 앞에 섭니다. 문을 열어본 적이 없습니다.

#### 선택지 (가짜)
```
[문을 연다]
[기다린다]
```

> [!IMPORTANT]
> 어떤 선택을 해도 결국 문을 열게 됩니다. 이는 **운명의 불가피성**을 암시합니다.

#### 대기 시 추가 연출
- 3초: 먼 곳에서 뭔가 쓰러지는 소리
- 10초: 숨소리 같은 것이 들림
- 20초: 자동으로 문이 살짝 열림

---

### 씬 5: 공허 속의 할아버지

#### 상황
문을 열자 충격적인 광경이 펼쳐집니다.

#### 연출 순서
1. 문이 열림
2. 바닥이 없음 — 보랏빛 공허가 펼쳐짐
3. 할아버지가 공중에 떠 있음 (쓰러진 자세)
4. 피를 흘리고 있음
5. 회중시계가 할아버지 손에서 떨어져 공중에 떠 있음

#### 시각적 묘사
```
┌─────────────────────────────────────────┐
│            [문틀]                        │
│     ┌─────────────────────┐             │
│     │     ●               │ ← 회중시계  │
│     │        ○            │ ← 할아버지  │
│     │       /|\           │   (부유)    │
│     │                     │             │
│     │  ≋≋≋≋≋≋≋≋≋≋≋≋≋≋≋  │ ← 보라색    │
│     │  ≋≋ 공 허 ≋≋≋≋≋≋  │    공허     │
│     └─────────────────────┘             │
└─────────────────────────────────────────┘
```

#### 크론의 행동
- 발을 내딛으려 하지만 바닥이 없음
- 할아버지를 잡으러 갈 수 없음
- 회중시계만 겨우 잡을 수 있음

---

### 씬 6: 문을 닫다

#### 상황
크론은 인기척이 위험하다고 판단하고, 조심스럽게 문을 닫습니다.

#### 대사 (독백)
```
크론: (떨리는 목소리) "할아버지... 할아버지?!"
크론: "...움직이면 안 돼. 누가 있을지도 몰라."
```

#### 연출
- 문을 닫을 때 "끼익" 소리에 긴장감 조성
- 문이 닫히자 방이 다시 "정상"으로 보임
- 하지만 창밖이 이상하게 변함

---

### 씬 7: 회중시계의 비밀

#### 상황
크론이 눈물을 흘리며 회중시계를 열어봅니다.

#### 발견
1. **시계바늘이 거꾸로 돌고 있음**
2. 창밖이 미친 듯이 변함 (밤↔낮 점멸)
3. 형체를 알 수 없는 보랏빛 소용돌이로 변함

#### 튜토리얼: 기어 발견
- 바닥에 떨어진(굴러다니던) **작은 톱니바퀴** 발견
- 문틀에 톱니바퀴를 끼울 수 있는 **슬롯** 발견

#### 🆕 첫 번째 퍼즐: 기어 조합

```
문제: 문을 열려면 올바른 기어 조합이 필요하다

[문틀 슬롯]
┌───┐   ┌───┐   ┌───┐
│ 1 │   │ 2 │   │ 3 │
│ ? │   │ ? │   │ ? │
└───┘   └───┘   └───┘

[보유 기어]
● 작은 기어 (바닥에서 발견)
◉ 중형 기어 (책상 서랍)
○ 큰 기어 (옷장 뒤)

[힌트]
책상 메모: "과거-현재-미래의 순서로..."
→ 큰-중형-작은 순서 = 과거에서 미래로 이동!
```

#### 상호작용
```
[E / A]: 톱니바퀴를 줍는다
[E / A]: 문틀 슬롯에 끼운다
[Tab / Select]: 기어 인벤토리 확인
```

---

### 씬 8: 시간의 정상화

#### 상황
톱니바퀴를 문틀에 끼우자 변화가 일어납니다.

#### 연출
1. 톱니바퀴가 돌아가기 시작
2. 창밖의 소용돌이가 멈춤
3. 색채가 돌아옴 (흑백 → 컬러)
4. 낮인지 밤인지 불명확한 황혼 상태

#### 내레이션
> *"톱니바퀴의 크기. 그것은 시간의 속도였다."*
> *"작은 톱니는 시간을 빠르게, 거대한 톱니는 시간을 느리게 흐르게 했다."*

---

### 씬 9: 다락방의 진실

#### 상황
크론은 깨닫습니다. 이 다락방이 단순한 방이 아니라는 것을.

#### 연출
- 벽, 천장, 바닥 아래에서 **거대한 기어**의 윤곽이 드러남
- 다락방 전체가 **시계 메커니즘**의 일부임을 시각화

#### 내레이션
> *"다락방은 단순한 창고가 아니었다."*
> *"째깍이는 소리는 작은 시계들이 내는 소음이 아니었다."*
> *"다락방 전체가 거대한 시계의 심장이었다."*

---

### 씬 10: 결의

#### 상황
크론은 할아버지를 찾기 위해 문을 다시 엽니다.

#### 🆕 추리 보드 도입
```
[발견] 할아버지의 수첩

크론: "할아버지의 수첩이다... 뭔가 적혀 있어."

수첩 내용:
- "시간은 되돌릴 수 있지만, 대가가 있다"
- "문틀 너머에는 다른 시간이 있다"
- "세 친구를 지켜봐라"
```

#### 추리 보드 UI 첫 등장
```
┌─────────────────────────────────────────────────┐
│              📋 크론의 추리 보드                 │
├─────────────────────────────────────────────────┤
│                                                 │
│  ┌────────┐           ┌────────┐               │
│  │ 단서 1 │    ???    │ 단서 2 │               │
│  │총소리🔊│           │할아버지│               │
│  └────────┘           └────────┘               │
│                                                 │
│  💡 핵심 질문: 할아버지에게 무슨 일이?         │
│                                                 │
└─────────────────────────────────────────────────┘
```

#### 대사
```
크론: "할아버지를 찾아야 해. 누가... 누가 이런 짓을 한 거야?"
크론: "범인을 찾아야 해. 이 수첩이 단서가 될 거야."
```

#### 연출
- 문을 열면 이번에는 **미래 도시**가 펼쳐짐
- 타이틀 로고 등장: **CHRONOS GEAR**
- 프롤로그 종료

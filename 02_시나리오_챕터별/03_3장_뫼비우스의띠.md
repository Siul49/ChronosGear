# 3장: 뫼비우스의 띠 (The Möbius Strip)

> *"거울 속에는 낯선 노인이 있었다. 그것이 나라는 걸 깨닫는 데 오랜 시간이 걸렸다."*

---

## 개요

| 항목 | 내용 |
|------|------|
| **플레이 시간** | 약 2-3시간 |
| **시간대** | 가변 (수십 년의 시간 여행) |
| **목적** | 정체성 반전 - 할아버지 = 미래의 자신 |
| **분위기** | 전율, 깨달음, 절망 |

---

## 3-1: 누적된 피로

### 상황
수많은 시간 여행으로 크론의 몸에 변화가 시작됩니다.

### 시각적 변화
```
[초기 크론]          [현재 크론]
   깔끔한 머리    →    흐트러진 머리
   맑은 눈        →    그늘진 눈
   아이 같은 뺨   →    살짝 움푹 들어감
```

### 게임플레이 반영
- 대시 거리 감소 (약간)
- TP 회복 속도 저하
- UI에 "피로도" 게이지 추가

### 다락방 귀환 시
```
거울 앞을 지나갈 때...
(거울 속 모습이 살짝 '더 늙어 보임')
크론: "...? 눈이 피곤해서 그런가."
```

---

## 3-2: 던전 - 영원의 순간

### 상황
안정화 파편을 더 모으기 위해 두 번째 던전 진입.

### 🆕 에코 클론 스킬 획득

던전 입구에서 새로운 능력을 발견합니다.

```
[시간이 멈춘 공간에서 회중시계가 공명]
[과거의 자신이 일시적으로 분리됨]

크론: "또 새로운 힘이... 내 분신?"
크론: "과거 몇 초의 나를 만들 수 있어!"

>> 에코 클론 스킬 해금! <<
[F]키로 3초 전의 자신을 소환합니다.
클론은 발판을 밟거나, 레버를 유지하거나, 적을 견제할 수 있습니다!
```

### 던전 특성
- 시간이 거의 멈춘 세계
- 물방울이 공중에 떠 있음
- 모든 것이 슬로우 모션

### 던전 구조

```
              [최종 보스: 영원의 수호자]
                        ▲
                        │ (동시 3발판 필요)
                        │
    ┌───────────────────┼───────────────────┐
    │                   │                   │
[정지 구역]        [허브 룸]          [파동 구역]
 열쇠 A             (안전)             열쇠 B
    │                   │                   │
    └───────────────[입구]──────────────────┘
                        │
                  [클론 구역]
                   열쇠 C
```

### 퍼즐 상세 (P201~P204 계열)

#### 정지 구역 (P001 응용)
| 퍼즐 | 설명 | 해법 |
|------|------|------|
| 정지 함정 | 들어가면 플레이어도 느려짐 | 시프트로 빠르게 탈출 |
| 멈춘 레버 | 당겨도 반응 없음 | 앵커로 시간 가속 후 당기기 |
| 투명 발판 | 정지 상태에서만 보임 | 앵커 on/off 반복 |

#### 파동 구역 (P102 응용)
| 퍼즐 | 설명 | 해법 |
|------|------|------|
| 타임 웨이브 | 주기적 시간 파동 | 리듬에 맞춰 회피 |
| 역류 기어 | 파동마다 방향 전환 | 시프트 + 타이밍 |
| 충격파 발판 | 파동이 발판 파괴 | 리와인드로 복원 |

#### 클론 구역 (P201~P203 핵심)
| 퍼즐 | 설명 | 해법 |
|------|------|------|
| 동시 스위치 | 2개 발판 동시 밟기 | 클론 배치 후 본체 이동 |
| 레버 유지 | 레버 잡고 있어야 문 유지 | 클론이 유지, 본체 통과 |
| 삼중 발판 | 3개 발판 동시 (보스 전) | 클론 2개 + 본체 |

### 🆕 조연 등장: 그레이

던전 내부에서 그레이 연구원을 만납니다.

```
[과거 레이어 - 연구 구역]

그레이: "벨로 박사! 이 주파수 패턴 좀 보세요!"
그레이: "화약 폭발이 아닙니다. 완전히 다른 거예요."
벨로: "음... 시간 입자 붕괴음 같은데?"
그레이: "그런 게 있어요? 대체 뭐가 터지면 이런 소리가..."

크론: (숨어서 듣는다) "시간 입자 붕괴음? 총성이 아니었어?"
크론: "저 사람이 뭔가 알고 있어. 미래에서 찾아봐야겠어."

>> 단서 획득: "그레이의 음파 분석" <<
```

### 보스: 영원의 수호자

시간이 멈춘 상태로 존재하는 고대 기사.

| 페이즈 | 보스 행동 | 퍼즐 해법 |
|--------|----------|----------|
| 1 | 3개 기어 보호막 | 클론 2개 + 본체로 동시 파괴 |
| 2 | 시간 역전 공격 | 리와인드로 공격 반사 |
| 3 | 무한 분열 | 정지 구역 유인 후 앵커로 고정 |

## 3-3: 모라의 부모님

### 상황
던전 탐험 중, 과거의 한 장면을 목격합니다.

### 목격 장면
```
[과거의 거리]
자동차가 달려오고 있음
어린 모라와 부모님이 횡단보도 앞에 있음
```

### 크론의 개입 (과거 기억)
```
(크론이 시간을 멈추려 시도)
자동차가 느려짐
하지만 그 순간 시공간에 균열 발생
어린 모라가 균열을 통해 '공허'를 목격
```

### 나비효과 연결
```
어린 모라: (눈이 커지며) "저게... 뭐야?"
어린 모라: "시간이... 멈춰 있어?"
```

> [!IMPORTANT]
> 크론이 모라를 구하려다가, 오히려 모라에게 "시간의 진실"을 보여준 것입니다.
> 이것이 현재 모라가 "시간 정지 신봉자"가 된 원인입니다.

## 3-4: 급격한 노화

### 상황
너무 많은 시간 조작으로 크론의 노화가 가속됩니다.

### 연출
```
[던전에서 복귀 후]
크론이 비틀거림
손을 보니 주름이 늘어남
머리카락에 흰색이 섞임
```

### 게임플레이 반영
- 이동 속도 15% 감소
- 공격 속도 10% 감소
- 하지만 TP 최대치 증가 (경험의 축적)

### 크론의 반응
```
크론: "왜 이러지? 몸이..."
크론: "시간을 쓸수록... 내 시간이 줄어드는 건가?"
```

## 3-5: 거울 속의 진실

### 상황
다락방의 거대 거울 앞에 선 크론.

### 연출 (핵심 씬)
```
1. 크론이 거울 앞에 섬
2. 거울 속에는... 백발의 노인
3. 주름진 얼굴, 지친 눈
4. 그런데 그 얼굴이... 익숙함
5. 죽은 할아버지의 얼굴
```

### 대사
```
크론: "...할아버지?"
크론: "아니, 잠깐. 왜 거울에..."
크론: "왜 내가..."

(긴 침묵)

크론: "...나야. 이게 나야."
크론: "할아버지가... 나였어?"
```

### BGM
- 모든 배경음 정지
- 심장 박동 소리만 울림
- 째깍 소리가 점점 느려짐

## 3-6: 회상 - 루프의 진실

### 상황
크론이 모든 것을 이해합니다.

### 플래시백 몽타주
```
[장면 1] 어린 크론이 회중시계를 줍는다
[장면 2] 청년 크론이 친구들을 조사한다
[장면 3] 중년 크론이 시간에 시달린다
[장면 4] 노년 크론이 다락방을 지킨다
[장면 5] 늙은 크론이... "그날" 쓰러진다
```

### 내레이션
> *"할아버지를 구하겠다는 일념으로 시간을 달렸던 모든 순간들이,"*
> *"사실은 그 비극적인 죽음의 순간을 향해 스스로 걸어가는 과정이었음을."*
> *"크론은 전율했다."*

---

## 3-7: 회중시계의 기원

### 상황
크론이 시계의 패러독스를 이해합니다.

### 자문자답
```
크론: "이 시계... 할아버지한테 받았어."
크론: "근데 할아버지가 나라면..."
크론: "내가... 나한테 준 거야?"
크론: "그럼 이 시계는 언제 만들어진 거지?"
크론: "처음이... 없어."
```

### 시각화
```
┌────────────────────────────────────┐
│     회중시계의 순환                 │
│                                    │
│   "태어남" ─────?────▶ "전달됨"   │
│       ▲                    │      │
│       │                    │      │
│       └────────────────────┘      │
│                                    │
│       시작점이 존재하지 않음        │
└────────────────────────────────────┘
```

## 3-8: 절망과 결의

### 상황
크론이 무너집니다.

### 대사
```
크론: "그럼 나는 뭘 한 거야?"
크론: "할아버지를 구하려고... 범인을 찾으려고..."
크론: "결국 내가 죽으러 가는 거잖아."
크론: "이 루프를 끊을 수 있어? 끊어야 하는 거야?"

(긴 침묵)

크론: "아니. 아직 답을 못 찾았어."
크론: "범인. 누가 나를 죽였어? 왜 죽였어?"
크론: "그걸 알아야... 뭐라도 바꿀 수 있어."
```

## 3-9: 챕터 엔딩 - "그날"로의 귀환

### 상황
크론은 결국 모든 시간선이 수렴하는 곳으로 가기로 결심합니다.

### 다락방 조작
```
[모든 기어 슬롯에 특수 기어 조합]
[회중시계가 격렬하게 진동]
[창밖이 모든 색으로 점멸]
```

### 목적지 설정
```
크론: "그날로 가야 해."
크론: "내가 죽는 그 순간으로."
크론: "거기서 범인을 만나야 해."
```

### 문을 열기 전
```
모라의 통신: "크론? 어디 가는 거야?"
크론: "모라... 나 이제 알 것 같아."
모라: "뭘?"
크론: "누가 범인인지."
모라: "...그래?"
크론: "근데 맞는지 확인해야 해."
모라: "...조심해, 크론."
```

## 챕터 요약

### 핵심 반전
- **할아버지 = 미래의 크론**
- 시간 조작의 대가 = 노화
- 회중시계 = 시작 없는 패러독스

### 게임플레이 변화
- 크론의 외형이 늙어감
- 스탯 변화 (속도↓, TP↑)
- 던전 난이도 상승

### 스토리 진행
- 자기 정체성 인식
- 루프의 필연성 이해
- 범인 추적의 최종 단계

### 획득 아이템
- 영원의 순간 파편
- 노화된 크론의 유물 해금
- 과거 기억 조각

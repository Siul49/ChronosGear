# 2장: 인과율의 오류 (The Error of Causality)

> *"과거를 바꿀 때마다 세계는 조금씩 어긋났다. 그리고 그 책임은 오롯이 나에게 있었다."*

---

## 개요

| 항목 | 내용 |
|------|------|
| **플레이 시간** | 약 3-4시간 |
| **시간대** | Year 10 ↔ Year 25 (과거↔미래 왕복) |
| **목적** | 나비효과 발견, 자책의 시작, 제4의 기어 존재 인지 |
| **분위기** | 혼란, 죄책감, 스케일 확장 |

## 2-1: 과거로의 첫 여행

### 다락방 조작
```
[기어 슬롯에 '역방향 톱니바퀴' 장착]
[시계바늘이 반대로 회전]
[창밖이 세피아 톤으로 변화]
```

### 과거 마을 도착 (Year 10, 사건 당일 아침)
- 평화로운 마을 풍경
- 시계 가게가 멀쩡함
- 어린 친구들이 학교로 향하는 모습

### 크론의 감상
```
크론: "이게... 그날 아침이야? 아직 아무 일도..."
크론: "할아버지를 구할 수 있어. 미리 막으면 돼!"
```

## 2-2: 첫 번째 개입 - 레이

### 상황
골목에서 어린 레이가 불량배에게 맞고 있습니다.

### 선택지
```
[1] 레이를 도와준다
[2] 지켜본다
```

### 선택 1: 도와준다
```
크론: (불량배를 물리침)
어린 레이: "고, 고마워! 너 누구야?"
크론: "그냥... 지나가던 사람이야."
```

### 선택 2: 지켜본다
```
(어린 레이가 결국 스스로 일어섬)
어린 레이: "다음엔... 다음엔 내가 이길 거야!"
```

> [!IMPORTANT]
> **핵심 메커니즘**: 어떤 선택을 해도 결과가 반영됩니다.
> 하지만 이미 "다른 크론"이 개입한 타임라인이 존재합니다.

## 2-3: 두 번째 개입 - 벨로

### 상황
어린 벨로가 첫 번째 실험에서 실패하려 합니다.

### 장면
```
어린 벨로: "이번엔 성공할 거야...!"
(폭발 직전)
```

### 선택지
```
[1] 시간을 되돌려 실패를 막는다
[2] 그대로 둔다
```

### 선택의 결과

| 선택 | 즉각적 결과 | 미래 영향 |
|------|------------|-----------|
| 막아줌 | 벨로가 성공을 경험 | 오만한 과학자 |
| 내버려둠 | 벨로가 좌절함 | 겸손한 연구자 (대체 타임라인) |

## 2-4: 미래로 복귀 - 이상 현상 발견

### 다락방 귀환
- 창밖이 네온 톤(미래)으로 변함
- 하지만 뭔가 이상함

### 달라진 미래
```
크론: "레이? 레이를 찾고 있는데..."
NPC: "레이 형사? 그 사람 3년 전에 순직했는데요?"
크론: "뭐?!"
```

또는

```
NPC: "벨로 박사? 그 미친 놈 지금 감옥에 있어요."
크론: "감옥? 무슨 죄로?"
NPC: "인체 실험으로 사람을 죽였으니까요."
```

### 크론의 혼란
```
크론: "내가... 뭘 바꾼 거야?"
크론: "과거를 건드리면 이렇게 되는 거야?"
```

## 2-5: 기어 사이의 세계 - 첫 진입

### 상황
미래가 너무 바뀌어서 다락방 연결이 불안정해집니다.

### 다락방 이상
```
[경고: 시간선 불일치 감지]
[안정화를 위해 기어 파편 필요]
```

### 🆕 리와인드 스킬 획득

```
[다락방 중앙에 새로운 빛이 출현]
[회중시계가 반응]

크론: "이건... 뭐지? 시계에서 뭔가..."
크론: "시간을... 되돌릴 수 있어!"

>> 리와인드 스킬 해금! <<
[R]키로 오브젝트의 시간을 되돌립니다.
부서진 것을 복원하거나, 움직이는 것을 원래 위치로!
```

### 던전 진입: 쇠락의 정원
- 시간이 너무 빨리 흐르는 세계
- 모든 것이 부식되고 풍화됨
- 여기서 **안정화 기어 파편** 획득 필요

### 던전 구조

```
                    [보스 룸]
                        ▲
                        │ (3개 열쇠 필요)
                        │
    ┌───────────────────┼───────────────────┐
    │                   │                   │
[A: 노화 구역]     [허브 룸]           [B: 가속 구역]
  열쇠 A 획득        (안전)             열쇠 B 획득
    │                   │                   │
    │                   │                   │
    └───────────────[입구]──────────────────┘
                        │
                   [C: 복원 구역]
                    열쇠 C 획득
```

### 퍼즐 상세

#### A 구역: 노화 구역 (P101 계열)
| 퍼즐 | 설명 | 해법 |
|------|------|------|
| 빠른 기어 발판 | 발판이 너무 빨리 회전 | 앵커로 감속 후 통과 |
| 노화된 문 | 시간 경과로 잠김 | 리와인드로 복원 후 열기 |
| 가속 레이저 | 빠르게 움직이는 레이저 | 시프트로 타이밍 통과 |

#### B 구역: 가속 구역 (P102 계열)
| 퍼즐 | 설명 | 해법 |
|------|------|------|
| 동기화 기어 | 두 기어 속도 맞추기 | 앵커로 한쪽 감속 |
| 소멸하는 발판 | 밟으면 빠르게 사라짐 | 시프트로 빠르게 이동 |
| 시간 폭탄 | 3초 후 폭발 | 리와인드로 타이머 리셋 |

#### C 구역: 복원 구역 (P103 계열)
| 퍼즐 | 설명 | 해법 |
|------|------|------|
| 부서진 다리 | 통과 불가 | 리와인드로 복원 |
| 시든 나무 | 길을 막고 있음 | 리와인드로 살려 비켜가기 |
| 연쇄 복원 | 레버→발판→문 30초 제한 | 빠른 리와인드 연속 |

### 🆕 조연 등장: 오리 의사

던전 중간에 과거의 오리 의사를 목격합니다.

```
[과거 레이어 전환]

(오리가 누군가를 치료 중)
오리: "이 상처는... 대체 뭐지? 총상이 아니야."
오리: "안에서 바깥으로 터진 것 같아..."

크론: (숨어서 듣는다) "저 사람이 할아버지를 진료한 의사?"
크론: "뭔가 알고 있는 것 같아. 나중에 찾아봐야겠어."

>> 단서 획득: "오리 의사의 의문" <<
```

## 2-6: 벨로의 경고

### 상황
미래로 돌아온 크론이 벨로를 다시 찾습니다.

### 벨로의 발견
```
벨로: "크론! 드디어 왔군. 내가 뭘 발견했는지 알아?"
벨로: "누군가 시간을 인위적으로 '편집'하고 있어."
크론: "편집? 무슨 뜻이야?"
벨로: "마치... 비디오를 자르고 붙이듯이."
벨로: "그리고 그걸 할 수 있으려면..."
벨로: "톱니바퀴가 하나 더 있어야 해. 제4의 기어!"
```

### 핵심 정보 공개
```
벨로: "다락방에는 슬롯이 3개 있다고 했지?"
벨로: "과거, 현재, 미래. 각각 하나씩."
벨로: "근데 4번째 슬롯이 있다면?"
벨로: "그건... 시간을 '정지'시키는 기어야."
```

## 2-7: 나비효과의 진실

### 상황
크론이 자신의 개입 결과를 깨닫습니다.

### 레이의 진실
```
(과거 개입 선택에 따라)

[도와준 경우]
레이: "...난 항상 누군가가 도와주길 기다렸어."
레이: "스스로 강해질 필요가 없었으니까."
레이: "그래서 지금... 이렇게 된 거야."

[내버려둔 경우]
레이: "그때 혼자 이겨냈어. 아무도 없었거든."
레이: "덕분에 진짜 형사가 됐지."
```

### 벨로의 진실
```
[막아준 경우]
벨로: "난 실패해본 적이 없어."
벨로: "그래서... 내가 틀릴 수 있다는 걸 몰랐어."

[내버려둔 경우]
벨로: "그 실패 덕분에 겸손해질 수 있었어."
벨로: "한계를 아는 과학자가 된 거지."
```

### 크론의 자책
```
크론: "내가... 친구들을 망친 거야?"
크론: "도와주려고 한 건데..."
크론: "선의가... 이런 결과를 낳은 거야?"
```

## 2-9: 챕터 엔딩 - 모라의 힌트

### 다락방 귀환
크론이 지쳐서 다락방으로 돌아옵니다.

### 모라의 통신
```
모라: "크론, 무슨 일이야? 피곤해 보여."
크론: "모라... 나 때문이야. 다 내가 망친 거야."
모라: "무슨 소리야? 자세히 말해봐."
크론: "과거를 바꾸려고... 친구들을 도우려고..."
모라: "..."
모라: "크론, 과거를 바꾸면 안 돼. 알지?"
모라: "시간은... 흘러야 해. 그게 자연스러운 거야."
```

### 복선 대사
```
모라: "아니, 잠깐. 정말 흘러야 하는 걸까?"
모라: "만약 시간을 멈출 수 있다면..."
모라: "더 이상 아무도 다치지 않을 텐데."
크론: "모라...?"
모라: (웃음) "아무것도 아니야. 쉬어."
```

## 챕터 요약

### 게임플레이 요소
- 과거↔미래 왕복 시스템 도입
- 선택에 따른 결과 차이 체험
- 첫 번째 던전(쇠락의 정원) 클리어

### 스토리 진행
- 나비효과 인지: 선의가 비극을 낳음
- 제4의 기어(정지 기어) 존재 힌트
- 모라의 진짜 생각 복선

### 획득 아이템
- 안정화 기어 파편
- 벨로의 깨진 안경 (유물)
- 시간 입자 연구 자료

### 해금 요소
- 영원의 순간 (2번째 던전) 진입 가능
- 리와인드 스킬 강화

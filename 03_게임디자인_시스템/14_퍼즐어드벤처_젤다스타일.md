# 퍼즐 어드벤처 게임 설계 (Puzzle Adventure Design)

> 젤다의 전설 스타일 추리/퍼즐/어드벤처 게임 설계 철학

---

## 왜 퍼즐 어드벤처인가? (Design Rationale)

### 전투 중심 → 퍼즐 중심 전환 이유

| 이유 | 설명 |
|------|------|
| **시간 능력의 본질** | 크로노 기어의 시간 조작 스킬은 본질적으로 **퍼즐 해결 도구**. 전투보다 퍼즐에서 더 창의적 활용 가능 |
| **스토리 적합성** | 추리/미스터리 스토리에 **단서 수집과 논리적 추론**이 더 어울림 |
| **차별화** | 시간 조작 액션 게임은 많지만, 시간 조작 **퍼즐 어드벤처**는 독특한 포지션 |
| **리플레이 가치** | 퍼즐은 해답 발견의 **"아하" 순간**을 제공, 전투보다 기억에 남음 |
| **접근성** | 액션 스킬이 부족해도 **논리력으로 클리어** 가능 → 더 넓은 타겟층 |

### 젤다 스타일 채택 이유

| 요소 | 젤다에서 배우는 점 | Chronos Gear 적용 |
|------|-------------------|------------------|
| **열쇠-잠금** | 진행의 명확한 목표 제시 | 기어 = 열쇠, 문틀 = 잠금 |
| **던전 아이템** | 새 능력으로 이전 구역 재탐색 | 시간 스킬 = 던전 아이템 |
| **퍼즐 보스** | 패턴 암기보다 약점 발견 | 시간 능력으로 약점 노출 |
| **비선형 탐험** | 자유로운 탐색 보상 | 숨겨진 단서/기억 파편 |

> [!IMPORTANT]
> **핵심 결론**: 시간 조작이라는 독특한 메카닉은 **전투보다 퍼즐에서 더 빛난다**.
> 플레이어가 "내가 직접 해법을 찾았다"는 성취감을 느끼게 하는 것이 목표.

## 게임 설계 철학

### 핵심 원칙
```
"전투는 퍼즐의 일부이고, 퍼즐은 스토리의 일부다"
```

| 원칙 | 설명 |
|------|------|
| **발견의 기쁨** | 플레이어가 스스로 해답을 찾았다고 느끼게 |
| **일관된 규칙** | 시간 능력이 모든 상황에서 동일하게 작동 |
| **복합적 해결** | 여러 능력을 조합해야 풀리는 퍼즐 |
| **점진적 난이도** | 새 메카닉 소개 → 연습 → 응용 → 변형 |

## 게임플레이 비중

### 기존 (전투 중심) vs 신규 (퍼즐 중심)

```
[기존]                        [신규]
┌────────────────────┐       ┌────────────────────┐
│ ████████████ 60%  │ 전투  │ ██████ 25%        │
│ ████ 20%          │ 퍼즐  │ ████████████ 45%  │
│ ████ 20%          │ 탐험  │ ██████ 30%        │
└────────────────────┘       └────────────────────┘
```

### 각 요소의 역할

| 요소 | 비중 | 역할 |
|------|------|------|
| **퍼즐** | 45% | 진행의 핵심, 시간 능력 활용 |
| **탐험** | 30% | 단서 수집, 세계관 발견, NPC 대화 |
| **전투** | 25% | 퍼즐 해결의 일부, 보스는 퍼즐 보스 |

## 던전 구조: 열쇠-잠금 시스템

### 젤다 스타일 던전 흐름

```
                    [던전 입구]
                         │
                    ┌────▼────┐
                    │  허브룸  │ ← 중앙 허브 (항상 접근 가능)
                    └────┬────┘
           ┌─────────────┼─────────────┐
           │             │             │
      ┌────▼────┐   ┌────▼────┐   ┌────▼────┐
      │ A 구역  │   │ B 구역  │   │ C 구역  │
      │ 열쇠 A  │   │ (잠금)  │   │ (잠금)  │
      └────┬────┘   └────▲────┘   └────▲────┘
           │             │             │
      [열쇠 A]──────→[A로 열기]        │
                         │             │
                    ┌────▼────┐        │
                    │ B 구역  │        │
                    │ 열쇠 B  │────────┘
                    └────┬────┘
                         │
                    ┌────▼────┐
                    │중간보스룸│ ← 던전 아이템 획득
                    └────┬────┘
                         │
               [던전 아이템으로 새 구역]
                         │
                    ┌────▼────┐
                    │최종보스룸│ ← 퍼즐 + 전투
                    └─────────┘
```

### 던전 진행 원칙

| 원칙 | 설명 |
|------|------|
| **백트래킹** | 새 아이템으로 이전 구역 재탐색 |
| **숏컷** | 진행 후 허브로 빠르게 복귀 가능 |
| **명확한 목표** | 어디로 가야 하는지 항상 힌트 존재 |
| **선택적 탐험** | 필수 경로 외 보상 구역 존재 |

## 시간 스킬의 퍼즐 활용

### 스킬별 퍼즐 유형

#### 크로노 시프트 (순간이동)
| 퍼즐 유형 | 설명 |
|-----------|------|
| 거리 퍼즐 | 점프로 도달 불가능한 발판 |
| 타이밍 퍼즐 | 움직이는 발판에 착지 |
| 레이저 회피 | 레이저 사이를 통과 |
| 스위치 연계 | 빠르게 여러 스위치 활성화 |

```
예시: 거리 퍼즐
[시작] ────── 3유닛 ────── [발판]
         ↑
    시프트로만 도달 가능
```

#### 타임 앵커 (영역 감속)
| 퍼즐 유형 | 설명 |
|-----------|------|
| 동기화 퍼즐 | 빠른 것과 느린 것을 맞춤 |
| 투사체 조작 | 총알을 느리게 해서 통과 |
| 적 고정 | 적을 느리게 해서 발판으로 사용 |
| 시간차 스위치 | 감속으로 타이밍 조절 |

```
예시: 동기화 퍼즐
[빠른 기어] ───○───○───○─── (1초 주기)
[느린 기어] ───────○─────── (3초 주기)

앵커로 빠른 기어를 감속 → 동기화 → 문 열림
```

#### 리와인드 (시간 되돌리기)
| 퍼즐 유형 | 설명 |
|-----------|------|
| 복원 퍼즐 | 부서진 다리/문 복원 |
| 상태 롤백 | 스위치 상태를 과거로 |
| 투사체 반사 | 적 공격을 되돌려 보냄 |
| 연쇄 복원 | 여러 오브젝트 순서대로 복원 |

```
예시: 복원 퍼즐
[파괴된 다리]          [리와인드]          [복원된 다리]
   ×  ×  ×      ──────────────▶      ═══════════
```

#### 에코 클론 (과거 분신)
| 퍼즐 유형 | 설명 |
|-----------|------|
| 동시 발판 | 두 스위치 동시 활성화 |
| 분신 협력 | 클론이 레버를 당기고 있는 동안 통과 |
| 시간차 공격 | 클론과 함께 보스 약점 노출 |
| 경로 분담 | 두 경로를 동시에 진행 |

```
예시: 동시 발판
      [문]
       ↑
[A 발판]   [B 발판]
   ↑          ↑
  클론      플레이어
```

## 퍼즐 난이도 설계

### 4단계 학습 곡선

```
[소개] → [연습] → [응용] → [변형]
  │         │        │        │
  ▼         ▼        ▼        ▼
단순 사용  반복    조합     창의력
```

| 단계 | 예시 (리와인드) |
|------|-----------------|
| **소개** | "부서진 상자를 복원해보세요" |
| **연습** | 여러 상자를 복원하며 이동 |
| **응용** | 복원 + 시프트 조합 |
| **변형** | 복원하면 적이 부활 (트레이드오프) |

## 보스 디자인: 퍼즐 보스

### 기존 vs 신규 보스 철학

| 항목 | 기존 (패턴 보스) | 신규 (퍼즐 보스) |
|------|-----------------|-----------------|
| 승리 조건 | HP 0으로 만들기 | **약점 해법 발견** |
| 공격 방식 | 패턴 외우고 회피 | **환경 이용** |
| 핵심 재미 | 액션 스킬 | **논리적 추론** |
| 페이즈 | HP 기반 | **해법 적용 기반** |

### 퍼즐 보스 구조

```
[Phase 1]
  - 보스 무적 상태
  - 환경 관찰 → 약점 힌트 발견
  - 시간 스킬로 약점 노출

[Phase 2]
  - 노출된 약점 공격
  - 피해 축적
  - 새로운 패턴 추가

[Phase 3]
  - 보스 반격
  - 더 복잡한 퍼즐
  - 최종 해법 적용 → 처치
```

### 예시: 시간 먹는 자 (1장 보스)

| 페이즈 | 상태 | 해법 |
|--------|------|------|
| 1 | 투사체 흡수, 무적 | **리와인드**로 투사체 반사 |
| 2 | 빠른 이동, 공격 불가 | **앵커**로 감속 후 공격 |
| 3 | 분열 | **클론**으로 양쪽 동시 공격 |

## 탐험 설계

### 탐험 보상 구조

| 발견 유형 | 보상 |
|-----------|------|
| **숨겨진 방** | 시간 파편 대량 |
| **NPC 대화** | 스토리 힌트, 사이드퀘스트 |
| **환경 단서** | 추리 보드 업데이트 |
| **시간 조각** | 능력 강화 |
| **기억의 파편** | 히든 엔딩 조건 |

### 탐험 유도 설계

```
[명확한 힌트]
  - 벽의 균열 → 리와인드 가능
  - 멀리 보이는 발판 → 시프트 필요
  - 눈에 띄게 다른 색상 → 조사 가능

[보상 시각화]
  - 상자/보물이 보이지만 접근 불가
  - "어떻게 저기에 갈까?" 유도

[지도 시스템]
  - 탐험한 방/안 한 방 구분
  - 미발견 아이템 표시 (나중에 해금)
```

## 추리 요소 통합

### 단서 수집 방식

| 방법 | 예시 |
|------|------|
| **오브젝트 조사** | 책상 위 메모, 벽의 흔적 |
| **NPC 대화** | 증언 수집 |
| **시간 이동** | 과거/미래 같은 장소 비교 |
| **퍼즐 해결** | 숨겨진 방에서 핵심 단서 |

## 챕터별 핵심 퍼즐/미스터리

| 챕터 | 핵심 미스터리 | 핵심 퍼즐 | 습득 능력 |
|------|--------------|-----------|-----------|
| 프롤로그 | 할아버지는 어디? | 기어 조합 | 기본 조작 |
| 1장 | 총소리의 출처 | 시간 레이어 | 크로노 시프트 |
| 2장 | 과거 개입의 증거 | 리와인드 연쇄 | 타임 앵커 |
| 3장 | 진짜 범인의 정체 | 클론 협력 | 리와인드 |
| 4장 | 시간을 멈추는 이유 | 모든 능력 종합 | 에코 클론 |

# 적 & 보스 설계 (Enemies & Bosses)

> *"시간의 빈틈에서 태어난 존재들. 그들도 한때는 무언가였다."*

---

## 1. 일반 적

### A. 틱톡커 (Tick-Tocker)

| 항목 | 내용 |
|------|------|
| **외형** | 머리가 메트로놈 / 시계추로 된 인형 |
| **HP** | 30 |
| **공격** | 박자 맞춰 돌진 |
| **약점** | 타임 앵커 (박자 깨짐) |

#### 패턴
```
틱(준비) → 톡(돌진)
    ↓
[타이밍 예측 가능]
```

---

### B. 과거의 망령 (Echo Specter)

| 항목 | 내용 |
|------|------|
| **외형** | 기사 갑옷 + 현대 무기 (시대 혼합) |
| **HP** | 60 |
| **공격** | 검술 + 총격 혼합 |
| **약점** | 리와인드 (투사체 반사) |

#### 패턴
```
[근접] 검 3회 베기 → [원거리] 총 2발 → 대기
```

---

### C. 보이드 글리치 (Void Glitch)

| 항목 | 내용 |
|------|------|
| **외형** | 보랏빛 에너지 덩어리, 형체 가변 |
| **HP** | 45 |
| **공격** | 순간이동 후 폭발 |
| **약점** | 시간 스킬만 데미지 (물리 면역) |

#### 특수 기믹
```
- 일반 공격 무효
- 타임 앵커 : 형체 고정 → 공격 가능
- 리와인드 : 과거 형태로 복원 → 약화
```

---

### D. 시간 먹는 자 (Time Devourer)

| 항목 | 내용 |
|------|------|
| **외형** | 입이 거대한 톱니바퀴 괴물 |
| **HP** | 100 |
| **공격** | 시간 흡수, 폭발 |
| **약점** | 크로노 시프트 (흡수 회피) |

#### 패턴
```
[흡수] 주변 시간 느려짐 → [축적] 게이지 상승 → [폭발] 광역 데미지
```

## 3. 메인 보스

| 항목 | 내용 |
|------|------|
| **HP** | 1000 |
| **등장** | 영원의 순간 던전 끝 |
| **외형** | 시간에 갇힌 고대 기사, 거대한 검 |

#### 페이즈 1: 느린 검격
| 패턴 | 설명 | 대응 |
|------|------|------|
| 대검 내려치기 | 느리지만 광범위 | 시프트로 옆으로 |

#### 페이즈 2: 시간 해방

### 4장 최종 보스: 심판자 모라 (Mora, The Judge)

| 항목 | 내용 |
|------|------|
| **HP** | 1500 |
| **페이즈** | 3 |
| **등장** | 4장 "그날"의 다락방 |

#### 페이즈 1: 검사 모라
```
공격 패턴:
- 정조준 사격 (빨간 레이저 예고)
- 부채꼴 난사
- 수류탄 투척

대응:
- 시프트 회피
- 앵커로 총알 감속
- 리와인드로 수류탄 반사
```

#### 페이즈 2: 심판자 모라
```
(제4의 기어 활성화)

공격 패턴:
- 시간 정지 구역 생성
- 순간이동 사격
- 연속 정지 공격

대응:
- 정지 구역 피하기
- 에코 클론으로 위치 파악
- 빠른 판단력 필요
```

#### 페이즈 3: 최후의 모라
```
(마스터 기어 파괴 시도)

환경 변화:
- 공허 침식 (바닥 소멸)
- 중력 불안정 (부유)

공격 패턴:
- 마스터 기어에 사격
- 공허 파동 발사
- 최후의 선언

승리 조건:
- 모라의 HP 0 또는
- 모라가 기어 파괴 전 저지
```

## 4. 보스 디자인 원칙

### 학습 곡선
```
1페이즈: 패턴 학습 (느리고 예측 가능)
2페이즈: 변형 (속도 증가 또는 패턴 추가)
3페이즈: 도전 (복합 패턴, 환경 변화)
```

### 시간 스킬 필수 활용
```
모든 보스는 특정 시간 스킬이 유리하도록 설계

틱톡커/시간먹는자 → 앵커
과거의 망령 → 리와인드
영원의 수호자 → 시프트
모라 → 모든 스킬 종합
```

### 스토리 연결
```
각 보스는 스토리와 직결

1장 보스: 레이의 어둠 (부패)
2장 보스: 벨로의 오만 (나비효과)
3장 보스: 시간의 무게 (노화)
4장 보스: 모라의 구원 (철학)
```

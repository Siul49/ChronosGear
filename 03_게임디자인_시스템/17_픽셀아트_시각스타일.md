# 픽셀 아트 스타일 가이드 (Pixel Art Style Guide)

> Chronos Gear 비주얼 스타일 및 아트 규격

---

## 왜 이 아트 스타일인가? (Art Rationale)

### 픽셀 아트 선택 이유

| 이유 | 설명 |
|------|------|
| **시간의 시각화** | 픽셀은 "디지털 시간의 단위"를 연상시킴, 시간 조작 게임에 적합 |
| **제작 효율** | 1인/소규모 팀에서 고퀄리티 아트 생산 가능 |
| **노스탤지어** | 레트로 감성이 "과거 여행" 테마와 시너지 |
| **쉐이더 호환** | 디더링/팔레트 스왑이 픽셀에서 가장 자연스러움 |
| **확장성** | 스프라이트 추가/수정이 3D보다 빠름 |

### "빅토리아 + 네온" 컨셉 선택 이유

| 요소 | 왜 이 조합인가? |
|------|----------------|
| **빅토리아 시계공방** | 기어/시계 = 시간의 물리적 표현, 스팀펑크 감성 |
| **네온 SF** | 미래 = 차가운 기술, 과거와의 대비 강조 |
| **두 시대의 충돌** | 게임의 핵심 = 과거↔미래 이동, 시각적으로 명확한 구분 |

### 32색 팔레트 제한 이유

| 이유 | 효과 |
|------|------|
| **일관성 강제** | 모든 아트가 동일한 색감 → 통일된 비주얼 |
| **시간대 스왑** | 팔레트 교체만으로 과거/미래 구분 가능 |
| **레트로 감성** | SNES/GBA 시대 느낌 재현 |
| **작업 속도** | 색상 고민 감소 → 아트 생산량 증가 |

> [!IMPORTANT]
> **핵심 원칙**: 모든 비주얼 요소는 **게임 메카닉을 지원**해야 함.
> 과거=따뜻함, 미래=차가움, 공허=불안함 → 플레이어가 직관적으로 이해.

## 1. 비주얼 컨셉

### 핵심 스타일
```
"빅토리아 시계공방 + 네온 SF"
```

| 요소 | 표현 |
|------|------|
| **시계/기어** | 복잡한 메카닉, 황동색 |
| **시간 왜곡** | 보라/파랑 발광 |
| **과거** | 따뜻한 세피아, 갈색 계열 |
| **미래** | 차가운 네온, 청록색 |
| **공허** | 깊은 보라, 검정 |

## 2. 픽셀 규격

### 해상도

| 항목 | 값 | 비고 |
|------|-----|------|
| **기준 해상도** | 320×180 | 16:9 비율 |
| **스케일 배율** | 6× | 1920×1080 출력 |
| **픽셀 크기** | 6×6 px | 화면 픽셀당 |

### 캐릭터 크기

| 오브젝트 | 픽셀 크기 | 실제 크기 |
|----------|-----------|-----------|
| 플레이어 | 16×24 px | 96×144 px |
| 일반 적 | 16×16 px | 96×96 px |
| 대형 적 | 32×32 px | 192×192 px |
| 보스 | 64×64 px | 384×384 px |
| NPC | 16×24 px | 96×144 px |

### 타일 크기
```
기본 타일: 16×16 px (96×96 실제)
대형 타일: 32×32 px (장식용)
```

### 4.1 크론 (주인공)

#### 노화 단계별 스프라이트

```
[어린 크론]     [청년 크론]     [중년 크론]     [노인 크론]
   ○               ○               ○               ○
  /█\             /█\             /█\             /▓\
  / \             / \             /▓\             /▓\
 호기심         결의             피로             체념
```

| 단계 | 머리 | 의상 | 특징 |
|------|------|------|------|
| 어린 | 갈색, 단정 | 밝은 셔츠 | 눈 크고 밝음 |
| 청년 | 갈색, 흐트러짐 | 코트 추가 | 눈빛 날카로움 |
| 중년 | 회색 섞임 | 낡은 코트 | 주름, 그늘 |
| 노인 | 백발 | 찢어진 코트 | 구부정, 느린 동작 |

## 5. 환경 디자인

### 5.1 다락방 (허브)

```
타일셋 구성:
┌────────────────────────────────────┐
│ [바닥] 나무판자 - 갈색 계열        │
│ [벽] 벽지 + 시계들 - 따뜻한 톤     │
│ [장식] 기어, 시계, 책 - 황동색     │
│ [문틀] 특수 - 보라색 발광          │
└────────────────────────────────────┘
```

### 5.2 미래 도시

```
타일셋 구성:
┌────────────────────────────────────┐
│ [바닥] 금속 그리드 - 청록 계열     │
│ [벽] 콘크리트 + 네온 - 차가운 톤   │
│ [장식] 홀로그램, 전선 - 발광       │
│ [배경] 빌딩 실루엣 - 어두운 톤     │
└────────────────────────────────────┘
```

### 5.3 과거 마을

```
타일셋 구성:
┌────────────────────────────────────┐
│ [바닥] 흙길, 돌길 - 세피아 계열    │
│ [벽] 나무 건물, 초가 - 따뜻한 톤   │
│ [장식] 꽃, 나무 - 자연색           │
│ [하늘] 일몰/일출 - 주황 그라데이션 │
└────────────────────────────────────┘
```

### 5.4 대시계 내부

```
타일셋 구성:
┌────────────────────────────────────┐
│ [바닥] 거대 기어 - 황동 + 검정     │
│ [벽] 기어 메커니즘 - 움직이는 타일 │
│ [장식] 진자, 태엽 - 발광 효과      │
│ [배경] 무한 기어 - 시차 스크롤     │
└────────────────────────────────────┘
```

## 7. 이펙트 스프라이트

### 시간 스킬 이펙트

| 스킬 | 이펙트 스타일 | 색상 |
|------|--------------|------|
| 크로노 시프트 | 잔상 + 글리치 | 파란 네온 |
| 타임 앵커 | 회전 기어 필드 | 보라 + 황동 |
| 리와인드 | 역방향 파티클 | 역재생 + 세피아 |
| 에코 클론 | 투명 잔상 | 반투명 파랑 |

### 피격/데미지 이펙트

| 유형 | 스타일 |
|------|--------|
| 일반 피격 | 흰색 플래시 + 추락 |
| 크리티컬 | 황금 플래시 + 슬로우 |
| 시간 피해 | 보라 글리치 |
| 회복 | 초록 파티클 상승 |

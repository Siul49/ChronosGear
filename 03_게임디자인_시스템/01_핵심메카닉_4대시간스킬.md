# 핵심 메커니즘 (Core Mechanics)

> *"시간은 너의 무기이자, 대가다."*

---

## 1. 기어 시스템 개요

### 핵심 컨셉
플레이어는 **시간을 직접 조작**하여 전투하고 퍼즐을 해결합니다.
모든 능력은 "회중시계"와 "톱니바퀴"라는 물리적 오브젝트에 기반합니다.

### 자원 관리

| 자원 | 용도 | 회복 방법 |
|------|------|-----------|
| **HP (존재력)** | 생명력 | 아이템, 리와인드 |
| **TP (시간력)** | 스킬 사용 | 공격 적중, 저스트 회피 |
| **기어 충전** | 궁극기 | 보스 처치, 특수 조건 |

| **타임 앵커** | 1장 1-3 | 벨로 연구소 진입 후 | 시간 왜곡 구역 접촉 시 해금 |

## 2. 4대 시간 스킬


### 스킬 A: 크로노 시프트 (Chrono Shift)

> 짧은 시간을 건너뛰어 순간이동

| 항목 | 내용 |
|------|------|
| **타입** | 이동/회피 |
| **소모** | TP 15 |
| **쿨타임** | 없음 (연속 사용 시 TP 추가 소모) |
| **조작** | 방향 + Shift (PC) / 방향 + RT (패드) |

#### 기본 효과
- 입력 방향으로 빠르게 대시
- 이동 중 무적 프레임 (0.2초)
- 잔상이 파란색 네온으로 남음

#### 저스트 회피 (Perfect Dodge)
```
조건: 적의 공격이 닿기 직전에 발동
보상: 
  - 화면이 흑백으로 반전
  - 2초간 불릿 타임 (적만 느려짐)
  - TP 30 회복
```

#### 시각 연출
```
[발동 전] ─→ [글리치 이펙트] ─→ [순간이동] ─→ [잔상 페이드]
   ○           ☆☆☆              ○→         ○ ┅┅○
```

### 스킬 B: 타임 앵커 (Time Anchor)

> 특정 영역의 시간을 느리게

| 항목 | 내용 |
|------|------|
| **타입** | 설치/디버프 |
| **소모** | TP 30 |
| **지속** | 4초 |
| **조작** | Q키 홀드 → 조준 → 해제 (PC) / LT 홀드 → 우스틱 → 해제 (패드) |

#### 기본 효과
- 지정 위치에 시계태엽 모양의 필드 설치
- 필드 내 적/투사체 속도 90% 감소
- 필드 내 아군은 영향 없음

#### 활용 예시
```
[투사체 방어]
적 총알 ──●──●──●──▶  [앵커 필드]  ──●─────────▶ (느려짐)
                     
[적 집중]
   ◆ ◆ ◆ (적들)  →  [앵커] → ◆◆◆ (뭉침) → 광역 공격!
```

### 스킬 C: 리와인드 (Rewind)

> 시간을 역재생하여 복원

| 항목 | 내용 |
|------|------|
| **타입** | 유틸리티/복원 |
| **소모** | TP 40 |
| **쿨타임** | 8초 |
| **조작** | E키 (PC) / LB (패드) |

#### 사용 대상별 효과

| 대상 | 효과 |
|------|------|
| **부서진 오브젝트** | 원래 상태로 복원 (다리, 스위치 등) |
| **적 투사체** | 발사자에게 되돌아감 (카운터 데미지) |
| **자기 자신** | 3초 전 상태로 복귀 (HP 포함) |
| **NPC/아이템** | 이전 상태로 복원 (퍼즐용) |

#### 자기 리와인드 제한
```
주의: 자기 자신에게 사용 시
  - 쿨타임 2배 (16초)
  - TP 2배 소모 (80)
  - 3장 이후 "노화 가속" 부작용 발생
```

---

### 스킬 D: 에코 클론 (Echo Clone)

> 과거의 잔상을 소환

| 항목 | 내용 |
|------|------|
| **타입** | 소환/협력 |
| **소모** | TP 50 |
| **지속** | 5초 |
| **조작** | R키 (PC) / RB (패드) |

#### 기본 효과
- 발동 시점 기준 **3초 전의 행동을 그대로 반복**하는 잔상 생성
- 잔상은 공격 가능 (데미지 50%)
- 잔상은 피격 불가 (투명 판정)

#### 활용 예시
```
[발판 퍼즐]
1. A 발판을 밟음
2. 에코 클론 생성 (잔상이 A를 밟음)
3. 플레이어는 B 발판으로 이동
4. 양쪽 동시 활성화 → 문 열림

[전투 활용]
1. 적 뒤로 이동하며 공격
2. 에코 클론 생성
3. 잔상이 적 뒤에서 공격 (협공)
4. 협공 보너스 데미지 +30%
```

## 3. 기어 슬롯 시스템

### 다락방 문틀의 슬롯
다락방 문틀에는 3개의 기어 슬롯이 있습니다.

```
┌─────────────────────────────────────┐
│              [문틀]                  │
│   ┌───┐     ┌───┐     ┌───┐        │
│   │ 1 │     │ 2 │     │ 3 │        │
│   │과거│     │현재│     │미래│        │
│   └───┘     └───┘     └───┘        │
│                                     │
│   슬롯 조합에 따라 이동 가능한 시간이  │
│   결정됩니다.                        │
└─────────────────────────────────────┘
```

### 기어 크기와 효과

| 기어 크기 | 시간 효과 | 용도 |
|-----------|-----------|------|
| **극소형** | 시간 급가속 | 미래로 크게 이동 |
| **소형** | 시간 가속 | 미래로 소폭 이동 |
| **중형** | 표준 속도 | 현재 유지 |
| **대형** | 시간 감속 | 과거로 소폭 이동 |
| **초대형** | 시간 급감속 | 과거로 크게 이동 |

### 조합 예시
```
[미래로 15년]
슬롯1: 대형 / 슬롯2: 중형 / 슬롯3: 소형

[과거로 5년]
슬롯1: 소형 / 슬롯2: 중형 / 슬롯3: 대형

[기어 사이의 세계]
슬롯1: 극소형 / 슬롯2: 없음 / 슬롯3: 초대형
```

## 4. 시간 조작의 대가

### 노화 시스템
시간 스킬을 사용할수록 크론은 **노화**합니다.

```
[노화 게이지]
░░░░░░░░░░ (0%)   - 어린 크론
▓▓▓▓░░░░░░ (40%)  - 청년 크론
▓▓▓▓▓▓▓░░░ (70%)  - 중년 크론
▓▓▓▓▓▓▓▓▓▓ (100%) - 노인 크론
```

### 노화 단계별 효과

| 단계 | 외형 | 이동속도 | 공격속도 | TP 최대치 | 특수 능력 |
|------|------|----------|----------|-----------|-----------|
| **어린** | 10세 | 100% | 100% | 100 | - |
| **청년** | 25세 | 95% | 95% | 130 | 위기 시 집중 |
| **중년** | 40세 | 85% | 85% | 160 | 경험의 통찰 |
| **노인** | 60세+ | 70% | 70% | 200 | 시간 초월 |

### 노화 회복
일부 특수 아이템으로 노화를 되돌릴 수 있지만, 스토리 진행에 따라 제한됩니다.

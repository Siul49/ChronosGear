## 6. 적 스폰 패턴

### 6.1 패턴 타입
| 패턴 | 설명 | 사용처 |
|------|------|--------|
| **동시** | 모든 적 즉시 스폰 | 소규모 조우 |
| **웨이브** | n초 간격 그룹 스폰 | 중규모 전투 |
| **트리거** | 플레이어 행동에 반응 | 이벤트 전투 |
| **무한** | 시간/체력 조건까지 지속 | 생존 구간 |

### 6.2 스폰 타이밍
```lua
-- 웨이브 예시 (LEVEL_1_002)
Wave_Config = {
    wave_1 = { enemies = {"Tiktok", "Tiktok"}, delay = 0 },
    wave_2 = { enemies = {"Glitch", "Glitch"}, delay = 5000 },
    victory_condition = "all_dead"
}
```

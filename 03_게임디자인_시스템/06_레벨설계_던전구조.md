# 레벨 디자인 (Level Design)

> *"공간은 시간의 그릇이다. 각 공간마다 시간이 다르게 흐른다."*

## 2. 허브: 다락방

### 기능
| 요소 | 기능 |
|------|------|
| **기어 슬롯** | 시간 이동 목적지 선택 |
| **거울** | 현재 노화 상태 확인 |
| **작업대** | 장비 강화/제작 |
| **기억의 책** | 수집한 정보 열람 |
| **침대** | HP/TP 완전 회복 |

### 레이아웃
```
┌─────────────────────────────────────┐
│           [창문]                     │
│    ┌───┐     ┌───┐                  │
│    │침대│     │거울│                  │
│    └───┘     └───┘                  │
│                                     │
│  [작업대]              [기억의 책]   │

│         ┌─────────┐                 │
│         │ 기어 슬롯 │ ← 3개의 슬롯   │

│         └────┬────┘                 │

```

## 3. 현실 세계 레벨

### A. 과거 마을 (Old Chronoville)

#### 분위기
- 따뜻한 색감 (세피아 톤)
- 평화로운 배경음
- 나비효과 개입 지점 표시

#### 주요 구역
| 구역 | 특징 | 이벤트 |
|------|------|--------|
| **학교** | 친구들의 어린 시절 | 추억 회상 |
| **골목** | 레이 관련 | 개입 선택지 |
| **벨로 창고** | 실험실 | 개입 선택지 |
| **모라 집 앞** | 거리 | 개입 선택지 |
| **시계 가게** | 크론의 집 | 스토리 핵심 |

#### 맵 구조
```
        [학교]
           │
    ┌──────┼──────┐
    │      │      │
[골목]──[광장]──[거리]
    │      │      │
    └──────┼──────┘
           │
      [시계 가게]
```

---

### B. 미래 도시 (Neo-Chronopolis)

#### 분위기
- 네온 + 안개
- 사이버펑크 + 스팀펑크
- 어두운 배경음

#### 주요 구역
| 구역 | 특징 | 관련 캐릭터 |
|------|------|-------------|
| **시장 지구** | 정보 수집 | NPC |
| **치안국** | 어두운 분위기 | 레이 |
| **연구소** | 위험 구역 | 벨로 |
| **법원 타워** | 높은 빌딩 | 모라 |
| **뒷골목** | 암시장, 단서 | 정보원 |

#### 맵 구조
```
              [법원 타워]
                  │
         ┌───────┼───────┐
         │       │       │
    [연구소]──[광장]──[치안국]
         │       │       │
         └───────┼───────┘
                 │
            [시장 지구]
                 │
            [뒷골목]
```

## 4. 던전: 기어 사이의 세계

### A. 쇠락의 정원 (Garden of Decay)

#### 컨셉
- 시간 가속 세계 (1초 = 100년)
- 모든 것이 풍화/부식
- 빠른 판단 요구

#### 기믹
| 기믹 | 설명 | 대응 |
|------|------|------|
| **노화 구역** | 서있으면 피로도 증가 | 빠르게 통과 |
| **부서지는 발판** | 밟으면 순식간에 부서짐 | 타임 앵커로 유지 |
| **시든 다리** | 통과 불가 | 리와인드로 복원 |

#### 구조 (3층)
```
[3층: 정원 심층부]
  - 보스: 쇠락의 정령
  - 안정화 파편 획득

[2층: 부식의 회랑]
  - 함정 다수
  - 퍼즐 구역

[1층: 입구의 정원]
  - 튜토리얼 구역
  - 기본 기믹 학습
```

---

### B. 영원의 순간 (The Eternal Moment)

#### 컨셉
- 시간 감속 세계 (100년 = 1초)
- 모든 것이 정지에 가까움
- 관찰과 인내 요구

#### 기믹
| 기믹 | 설명 | 대응 |
|------|------|------|
| **정지 구역** | 들어가면 플레이어도 느려짐 | 시프트로 탈출 |
| **떠있는 물방울** | 발판으로 활용 가능 | 점프 대용 |
| **느린 적** | 천천히 움직이나 강력 | 패턴 관찰 |

#### 구조 (3층)
```
[3층: 영원의 제단]
  - 보스: 영원의 수호자
  - 시간 파편 대량 획득

[2층: 멈춘 시간의 사원]
  - 협력 퍼즐 다수
  - 에코 클론 필수

[1층: 순간의 안뜰]
  - 기믹 학습
  - 탐험 구역
```

---

### C. 불안정 지대 (Unstable Zone)

#### 컨셉
- 시간 속도가 구역마다 다름
- 예측 불가능
- 적응력 시험

#### 기믹
| 기믹 | 설명 | 대응 |
|------|------|------|
| **가변 경계** | 지역마다 속도 다름 | 기억하고 이동 |
| **글리치 구역** | 랜덤 시간 변화 | 빠른 판단 |
| **동기화 발판** | 특정 속도에서만 활성화 | 정확한 구역 선택 |

#### 구조 (미로형)
```
┌───┬───┬───┐
│ S │ F │ S │  S = 느림
├───┼───┼───┤  F = 빠름
│ F │ ? │ F │  ? = 랜덤
├───┼───┼───┤
│ S │ F │ G │  G = 골
└───┴───┴───┘
```

---

### D. 공허의 심연 (The Void Abyss)

#### 컨셉
- 시간이 정의되지 않음
- 중력 없음, 공간 왜곡
- 최종 보스전 무대

#### 특징
| 특징 | 효과 |
|------|------|
| **무중력** | 부유 상태 이동 |
| **논리 붕괴** | 계단이 옆으로, 문이 바닥에 |
| **공허 침식** | 시간이 지나면 발판 소멸 |

#### 구조 (비선형)
```
    ┌───┐
    │ 3 │
    └─┬─┘
      │
  ┌───┼───┐
  │   │   │
┌─┴─┐ │ ┌─┴─┐
│ 1 │ │ │ 2 │
└───┘ │ └───┘
      ▼
   [보스]
```

## 5. 레벨 진행 가이드

### 전체 흐름
```
[프롤로그]
다락방(튜토리얼) → 시계 가게

[1장]
미래 도시: 시장→치안국→연구소→법원

[2장]
과거↔미래 왕복 + 쇠락의 정원

[3장]
영원의 순간 + 불안정 지대

[4장]
다락방(그날) → 공허의 심연 → 엔딩
```

### 예상 플레이 타임
| 챕터 | 시간 |
|------|------|
| 프롤로그 | 30분 |
| 1장 | 2-3시간 |
| 2장 | 3-4시간 |
| 3장 | 2-3시간 |
| 4장 | 1.5-2시간 |
| **총합** | **9-12시간** |

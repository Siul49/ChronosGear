# 주요 장소 (Locations)

> *"공간은 시간의 그릇이다. 그릇의 크기가 시간의 속도를 결정한다."*

## 현실 세계

### 시계 가게 (The Clockwork Shop)

> 모든 것의 시작점

#### 설정
| 항목 | 내용 |
|------|------|
| **위치** | 작은 마을의 골목 |
| **외관** | 낡은 나무 건물, 태엽 장식 간판 |
| **분위기** | 째깍거리는 수백 개의 시계 소리 |

#### 구조
```
[3층: 다락방] ← 게임의 핵심 공간
    │
[2층: 거주 공간] ← 크론의 방, 복도
    │
[1층: 시계 판매장] ← 손님 응대, 수리 작업대
```

#### 숨겨진 비밀
- 1층의 시계들은 모두 **가짜** (장식용)
- 진짜 "시간"은 다락방에서만 다룸
- 지하에는 **초기 기어 파편 보관소** 존재

---

### 다락방 (The Attic)

> 우주의 심장, 시간의 제어실

#### 설정
| 항목 | 내용 |
|------|------|
| **정체** | 대시계의 제어 장치가 있는 차원 접점 |
| **크기** | 겉보기엔 작지만 내부 공간이 왜곡됨 |
| **특성** | 외부 시간과 완전 분리 |

#### 핵심 오브젝트

| 오브젝트 | 기능 | 상호작용 |
|----------|------|----------|
| **문틀의 기어 슬롯** | 외부 시간 속도 조절 | 기어 삽입/제거 |
| **거대 거울** | 시간선 간 연결 | 진실의 순간에 활성화 |
| **태엽 감는 장치** | 우주 시간 유지 | 멈추면 세계 정지 |
| **달력 없는 시계** | 현재 위치 표시 | 읽기 전용 |

#### 시각적 연출
- 기어 장착 시: 벽면에 시계 문양이 빛남
- 기어 미장착 시: 창밖이 보랏빛 공허로 변함
- 위험 상황: 천장에서 먼지가 거꾸로 올라감

---

### 미래 도시 (Neo-Chronopolis)

> 15년 후의 세계, 스팀펑크 + 사이버펑크

#### 설정
| 항목 | 내용 |
|------|------|
| **시대** | Year 25 (사건 후 15년) |
| **분위기** | 네온과 증기가 공존 |
| **상태** | 번영하지만 어딘가 부패함 |

#### 주요 구역

| 구역 | 설명 | 관련 캐릭터 |
|------|------|-------------|
| **치안국** | 경찰 본부, 어두운 조명 | 레이 |
| **벨로 연구소** | 시간 입자 실험실, 위험 구역 | 벨로 |
| **법원 타워** | 모라의 집무실, 최상층 | 모라 |
| **뒷골목** | 정보 수집, 암시장 | NPC |

#### 시각적 테마
- **색상**: 청록색 네온 + 녹슨 금속
- **날씨**: 항상 안개 또는 산성비
- **소리**: 기계음, 먼 곳의 사이렌

### 과거 마을 (Old Chronoville)

> 크론의 어린 시절, 목가적이지만 그림자가 있음

#### 설정
| 항목 | 내용 |
|------|------|
| **시대** | Year 5-10 |
| **분위기** | 따뜻하지만 불안한 기운 |
| **상태** | 평화롭게 보이지만 시간 왜곡의 조짐 |

#### 주요 장소

| 장소 | 설명 | 이벤트 |
|------|------|--------|
| **학교 운동장** | 경찰 놀이를 하던 곳 | 회상, 나비효과 개입점 |
| **벨로의 창고** | 첫 실험 실패 장소 | 개입 선택지 |
| **골목** | 레이가 불량배에게 맞던 곳 | 개입 선택지 |
| **모라의 집 앞** | 부모님 사고 현장 | 개입 선택지 |

## 기어 사이의 세계 (Gear-Between)

다락방에서 특정 기어 조합으로 진입하는 **이세계 던전**입니다.

### 쇠락의 정원 (Garden of Decay)

> 시간이 너무 빨리 흐르는 세계

#### 설정
| 항목 | 내용 |
|------|------|
| **기어 크기** | 초대형 (시간 가속) |
| **시간 비율** | 1초 = 100년 |
| **위험** | 순식간에 노화, 장비 부식 |

#### 환경
- 모든 것이 풍화되어 먼지로 변함
- 식물이 피었다가 순식간에 시듦
- 움직이면 피로도가 급격히 상승

#### 공략 핵심
- **타임 앵커**로 안전 구역 생성
- 빠르게 이동하며 파편 수집
- 체력 회복 아이템 필수

---

### 영원의 순간 (The Eternal Moment)

> 시간이 거의 멈춘 세계

#### 설정
| 항목 | 내용 |
|------|------|
| **기어 크기** | 극소형 (시간 감속) |
| **시간 비율** | 100년 = 1초 |
| **위험** | 움직임이 극도로 느려짐 |

#### 환경
- 모든 것이 정지에 가까움
- 물방울이 공중에 떠있음
- 적들도 느리지만, 일단 맞으면 연속타

#### 공략 핵심
- **크로노 시프트**로 빠르게 이동
- 적의 공격이 느리므로 패턴 파악 용이
- 긴 쿨타임의 강력한 스킬 사용에 유리

---

### 불안정 지대 (Unstable Zone)

> 시간이 무작위로 변하는 세계

#### 설정
| 항목 | 내용 |
|------|------|
| **기어 크기** | 가변 (무작위) |
| **시간 비율** | 구역마다 다름 |
| **위험** | 예측 불가능 |

#### 환경
- 한 구역은 빠르게, 옆 구역은 느리게
- 경계선을 넘을 때마다 시간 속도 변화
- 시각적으로 공간이 "깨져" 보임

#### 공략 핵심
- **에코 클론**으로 정찰
- 구역별 시간 속도 기억
- 리듬감 있는 플레이 필요

---

### 공허의 심연 (The Void Abyss)

> 시간이 정의되지 않은 최종 구역

#### 설정
| 항목 | 내용 |
|------|------|
| **기어** | 없음 (연결 끊김) |
| **시간** | 정의 불가 |
| **위험** | 존재 자체가 사라짐 |

#### 환경
- 보랏빛 안개가 가득
- 중력 없음, 부유 상태
- 논리가 깨진 구조 (계단이 옆으로, 문이 바닥에)

#### 등장 시점
- 4장에서 모라가 마스터 기어를 파괴할 때
- 엔딩 분기의 최종 무대

## 장소별 맵 진행 순서

```
[프롤로그]
다락방 → 복도 → 거실 → 현관 (튜토리얼)
    │
    ▼
[1장]
미래 도시: 치안국 → 뒷골목 → 연구소 → 법원
    │
    ▼
[2장]
기어 사이: 쇠락의 정원 → 영원의 순간 → 불안정 지대
    │
    ▼
[3장]
과거 마을 ↔ 미래 도시 (왕복, 나비효과 퍼즐)
    │
    ▼
[4장]
다락방 → 공허의 심연 → 엔딩
```

## 핵심 요약

> [!IMPORTANT]
> - **다락방**: 모든 공간을 연결하는 허브
> - **현실 세계**: 시간에 따른 변화와 친구들의 타락
> - **기어 사이**: 시간 속도가 극단적으로 다른 던전
> - **공허**: 최종 보스전 무대, 시간이 정의되지 않음

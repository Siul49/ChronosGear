# 캐릭터 관계도 및 요약

> *"우리는 모두 시간의 피해자이자, 가해자였다."*

---

## 캐릭터 관계도

```
                    ┌─────────────┐
                    │    크론     │
                    │  (주인공)   │
                    └──────┬──────┘
           ┌───────────────┼───────────────┐
           │               │               │
           ▼               ▼               ▼
    ┌──────────┐    ┌──────────┐    ┌──────────┐
    │   모라   │    │   레이   │    │   벨로   │
    │ (심판자) │    │ (수호자) │    │ (탐구자) │
    └────┬─────┘    └────┬─────┘    └────┬─────┘
         │               │               │
         ▼               ▼               ▼
    시간을 멈춤      힘에 의존      금기를 넘음
    (정지의 구원)   (무력한 정의)   (오만한 지식)
```

### 크론과의 관계 변화

| 시점 | 모라 | 레이 | 벨로 |
|------|------|------|------|
| **어린 시절** | 친구 (신뢰) | 친구 (보호) | 친구 (존경) |
| **15년 후** | 조력자 (동맹) | 용의자 (의심) | 용의자 (경계) |
| **진실 후** | 적대 (대립) | 이해 (화해) | 협력 (속죄) |

---

## 캐릭터 핵심 요약

> [!IMPORTANT]
> - **크론**: 선의가 비극을 불러온 죄책감의 아이콘
> - **모라**: 틀린 방법으로 맞는 목적을 추구하는 그레이 캐릭터
> - **레이**: 도움받다가 자립심을 잃은 비극적 영웅
> - **벨로**: 실패를 모르고 자란 천재의 오만과 파멸

# 핵심 NPC - 세 친구 (용의자 겸 조력자)

> *"우리는 모두 시간의 피해자이자, 가해자였다."*

---

## 1. 모라 (MORA) - 최종 보스 / 거짓 조력자

> 정의를 위해 시간을 멈추려는 심판자

### 기본 정보

| 항목 | 내용 |
|------|------|
| **이름** | 모라 (Mora) |
| **나이** | 10세 (어린 시절) → 25세 (성인) |
| **역할** | 조력자 → 최종 보스 |
| **직업** | 엘리트 검사 / 치안 총수 |
| **상징** | 천칭, 권총 |

### 성격 변화 (단계별)

| 단계 | 시점 | 내면 변화 | 외부 행동 | 논리적 연결 |
|------|------|-----------|-----------|-------------|
| 1 | 어린 시절 | 부모님을 잃을 뻔한 공포 | 정의에 집착 시작 | 불안정한 것을 통제하고 싶음 |
| 2 | 크론의 개입 | 시간 왜곡 → 공허 목격 | 악몽, 불면증 시작 | "세상에는 숨겨진 법칙이 있다" 직감 |
| 3 | 청소년기 | "통제 불가"에 대한 공포 | **법과 규칙** 연구 | "규칙으로 혼돈을 막을 수 있다" |
| 4 | 법대 시절 | 법으로도 시간(죽음)은 못 막음 | 철학/물리학 관심 | "법보다 근본적인 것을 통제해야" |
| 5 | 벨로와 교류 | 시간 연구 접촉 → 종말 재목격 | 시간 정지 확신 | "시간을 멈추면 죽음도 없다" |
| 6 | 현재 | 크론 = 세계 파멸 원인 확신 | 동맹 가장 후 감시 | 크론 제거 or 시간 정지 준비 |

> **핵심 논리**: 공허 목격 → "통제 불가의 공포" → 법(규칙)으로 통제 시도 → 법의 한계 → 시간 자체 통제

### 동기 상세

모라는 시간이 계속 흐르면 도달하는 **필연적 종말**을 보았습니다:
- 크론이 시간을 남용하여 인과율이 완전히 붕괴
- 세계가 공허로 침식되어 소멸
- 유일한 해결책: **시간을 영원히 멈추는 것**

> [!NOTE]
> 모라의 관점에서 크론은 "세계를 파괴하는 시한폭탄"입니다.
> 그녀는 악당이 아닌, **다른 방식의 구원자**입니다.

### 실패한 해결책들 (시간 정지 이전의 시도)

모라는 시간 정지라는 극단적 결론에 도달하기 전, **다른 모든 방법을 시도**했습니다:

| 시도 | 시점 | 방법 | 결과 | 모라의 결론 |
|------|------|------|------|-------------|
| **크론 제거** | Year 22 | 시간 이동 전 암살 시도 | 크론이 루프로 부활 | "그는 시간의 일부다. 죽일 수 없다." |
| **회중시계 파괴** | Year 23 | 시계를 확보해 용해로에 투입 | 시계가 15분 후 재생 | "시계는 시간 자체다. 파괴 불가." |
| **다락방 폐쇄** | Year 24 | 다락방 입구를 콘크리트로 봉인 | 다른 장소에 새 문 출현 | "다락방은 공간이 아닌 개념이다." |
| **법적 규제** | Year 24 | 시간 범죄 특별법 제정 시도 | 시간 개입은 증거가 남지 않음 | "법으로는 시간을 다스릴 수 없다." |
| **벨로 협력** | Year 25 | 시간 역전 기술로 개입 취소 | 역전할수록 새 분기 생성 | "과거를 고치면 미래가 갈라진다." |

```
모라의 사고 흐름:
┌─────────────────────────────────────────────────────┐
│  "크론을 막아야 한다"                               │
│         ↓                                          │
│  크론 제거 → 실패 (불멸)                            │
│  시계 파괴 → 실패 (재생)                            │
│  다락방 봉인 → 실패 (이동)                          │
│  법적 규제 → 실패 (증거 없음)                       │
│  과거 수정 → 실패 (분기 증가)                       │
│         ↓                                          │
│  "흐르는 시간 안에서는 해결이 불가능하다"            │
│         ↓                                          │
│  ★ "그렇다면 시간 자체를 멈추면 된다" ★            │
└─────────────────────────────────────────────────────┘
```

> [!IMPORTANT]
> 모라는 **미친 것이 아닙니다**. 그녀는 3년에 걸쳐 모든 합리적 방법을 시도한 후,
> 논리적으로 "시간 정지"라는 결론에 도달했습니다.
> 이것이 그녀를 더 무서운 적으로 만듭니다.

### 대사 예시

> *"흐르는 것은 결국 썩어 문드러져. 나는 이 세계를 가장 아름다운 순간에 박제할 거야."*
> *"크론, 네가 진짜 세상을 구하고 싶다면... 멈춰."*
> *"난 이미 다 시도해봤어. 죽이고, 부수고, 막고, 법을 만들고. 아무것도 안 됐어."*

## 2. 레이 (RAY) - 용의자 1 / 나중 조력자

> 빛을 잃은 정의, 무력해진 수호자

### 기본 정보

| 항목 | 내용 |
|------|------|
| **이름** | 레이 (Ray) |
| **나이** | 10세 (어린 시절) → 25세 (성인) |
| **역할** | 용의자 1 → 조력자 |
| **직업** | 형사 (부패함) |
| **상징** | 권총, 낡은 배지 |

### 성격 변화 (단계별)

| 단계 | 시점 | 내면 변화 | 외부 행동 | 논리적 연결 |
|------|------|-----------|-----------|-------------|
| 1 | 어린 시절 | 불량배에게 시달림, 강해지고 싶음 | 매일 싸우지만 계속 짐 | "언젠가 이길 거야" |
| 2 | 크론의 개입 | 갑자기 불량배가 사라짐 → 안도 | 싸울 필요가 없어짐 | 승리 경험 박탈 |
| 3 | 청소년기 | "누군가 해결해주겠지" | 문제 회피, 책임 전가 | 의존성 형성 |
| 4 | 졸업 후 방황 | 뭘 해야 할지 모름 | **경찰 입직** | "어릴 때 정의로웠던 나"를 되찾고 싶음 |
| 5 | 신입 시절 | 현실의 부패에 충격 | 저항 시도 → 실패 | "내 힘으로는 못 바꿔" |
| 6 | 현재 | 자기혐오 + 체념 | 뇌물 수수, 술 의존 | "최소한 살아는 가야지" |

> **경찰 입직 이유**: 어릴 때 **경찰 놀이**를 하며 "정의로운 나"를 꿈꿨음 → 방황 끝에 유일하게 떠오른 꿈

### 서브플롯

- 크론은 처음에 레이가 범인(총소리)이라고 의심
- 조사 과정에서 레이의 진짜 고통을 알게 됨
- 결국 레이는 크론에게 "네가 구해준 거 고맙다고 생각했는데... 그게 아니었구나"라고 고백

### 대사 예시

> *"정의? 그딴 건 어린애들 놀이야. 세상은 힘 있는 놈이 이겨."*
> *"...그래도 그때, 네가 나를 도와줬잖아. 그건 진짜였어."*

## 3. 벨로 (VELO) - 용의자 2 / 기술 조력자

> 금기를 넘어선 천재, 시간을 탐하는 과학자

### 기본 정보

| 항목 | 내용 |
|------|------|
| **이름** | 벨로 (Velo) |
| **나이** | 10세 (어린 시절) → 25세 (성인) |
| **역할** | 용의자 2 → 양면적 조력자 |
| **직업** | 시간 입자 연구 과학자 |
| **상징** | 깨진 안경, 시계 파편 |

### 성격 변화 (단계별)

| 단계 | 시점 | 내면 변화 | 외부 행동 | 논리적 연결 |
|------|------|-----------|-----------|-------------|
| 1 | 어린 시절 | 호기심 많음, 실험 좋아함 | 첫 실험 도전 | **보통 수준의 재능** |
| 2 | 크론 개입 전 | 실패 직전 → 좌절 예정 | 폭발 직전 | 여기서 실패하면 겸손해졌을 것 |
| 3 | 크론의 개입 | 시간 되돌림 → 성공 | "내가 해냈다!" | **운이 실력으로 착각** |
| 4 | 청소년기 | 자신감 → 과감한 시도 | 연속 성공 | 크론이 **매번 몰래 개입** |
| 5 | 대학 시절 | "나는 천재, 실패는 없다" | 동료 무시, 윤리 경시 | 겸손 학습 기회 완전 상실 |
| 6 | 현재 | 금기에 손대도 된다는 확신 | 시간 입자 연구 | "내가 하면 성공한다" |

> **핵심 설정**: 벨로는 원래 **평범한 재능**, 크론이 여러 차례 몰래 실패를 막아줌  
> → 벨로 본인은 "나는 천재"라고 믿지만, **실제로는 크론이 만든 가짜 천재**  
> → 3장에서 이것이 밝혀지며 벨로의 정체성 붕괴

### 서브플롯

- 벨로는 "너무 쉬운 성공"에 공허함을 느꼈음
- 그래서 더 극단적인 도전(시간 제어)에 빠짐
- 크론과의 대면에서 "누군가 내 실험에 손을 댔다"는 것을 눈치챔

### 대사 예시

> *"누군가 시간을 인위적으로 '편집'하고 있어. 톱니바퀴가 하나 더 있다고!"*
> *"실패? 나는 실패해본 적이 없어. ...그게 문제인 줄도 몰랐지."*

# 세계관 설정 (World Setting)

> *"우주는 거대한 시계이며, 다락방은 그 심장이다."*

---

## 1. 우주의 구조: 대시계(The Grand Clock)

### 1.1 기본 설정
우주 전체는 **거대한 시계 메커니즘**으로 작동합니다. 일반인은 이를 인지하지 못하지만, 모든 물리법칙과 인과율은 "째깍"거리는 시간의 흐름 위에 세워져 있습니다.

```
┌─────────────────────────────────────────────────────────┐
│                    대 시 계 (The Grand Clock)            │
│  ┌─────────────┐                    ┌─────────────┐     │
│  │  대형 기어   │◀────맞물림────▶│  소형 기어   │     │
│  │ (느린 시간)  │                    │ (빠른 시간)  │     │
│  └─────────────┘                    └─────────────┘     │
│           ▲                              ▲              │
│           │                              │              │
│           └──────── 다락방 ─────────────┘              │
│                    (제어실)                              │
└─────────────────────────────────────────────────────────┘
```

### 1.2 두 개의 거대 톱니바퀴
우주에는 두 개의 **마스터 기어**가 존재합니다:

| 기어 | 특성 | 영역 |
|------|------|------|
| **대형 기어 (Greater Gear)** | 느린 시간, 묵직함, 영원에 가까움 | 별들의 수명, 지질학적 변화 |
| **소형 기어 (Lesser Gear)** | 빠른 시간, 가벼움, 찰나적 | 인간의 삶, 계절의 변화 |

이 두 기어가 맞물려 돌아가면서 우주의 시간이 흐릅니다.

---

## 2. 다락방: 우주의 심장

### 2.1 정체
겉보기에는 낡은 시계 가게의 다락방이지만, 실제로는 **대시계의 제어실**입니다.

- **위치**: 두 마스터 기어가 맞물리는 접점
- **기능**: 시간의 흐름을 조율하고 유지
- **수호자**: 크론의 혈통이 대대로 관리

### 2.2 다락방의 특성

#### 시간의 고립
다락방 내부는 외부 시간과 분리되어 있습니다. 밖에서 10년이 흘러도 안에서는 하루일 수 있습니다.

#### 문틀의 슬롯
다락방 문틀에는 톱니바퀴를 끼울 수 있는 **기어 슬롯**이 있습니다:
- 작은 기어 장착 → 외부 시간이 빠르게 흐름
- 큰 기어 장착 → 외부 시간이 느리게 흐름
- 기어 미장착 → 외부가 **공허(Void)**로 변함

#### 공허 (The Void)
시간이 정의되지 않은 보랏빛 공간. 기어 연결이 끊기면 다락방 밖이 공허로 변합니다.
- 중력 없음 (물체가 부유)
- 인과율 없음 (논리가 깨짐)
- 장시간 노출 시 존재 자체가 사라짐

## 3. 회중시계: 마스터 키

### 3.1 기원의 패러독스
회중시계의 기원은 **불명**입니다. 오래된 전설에서는 "누군가"가 만들었다고 하지만, 실제로는 시간 루프 안에서 **시작도 끝도 없이 존재**합니다.

```
┌────────────────────────────────────────────────┐
│           시간 루프 속 회중시계의 역설          │
│                                                │
│   어린 크론 ─(시계를 받음)─▶ 노인 크론        │
│       ▲                          │            │
│       │                          │            │
│       └───(시계를 떨어뜨림)◀─────┘            │
│                                                │
│   시계는 누가 처음 만들었는가? → 해답 없음     │
└────────────────────────────────────────────────┘
```

### 3.2 기능
- 다락방과 소유자를 연결하는 **열쇠**
- 소유자의 시간(수명)을 동력으로 사용
- 작은 규모의 시간 조작 가능 (리와인드, 슬로우 등)

### 3.3 대가
시계를 사용할수록 소유자는 **노화**합니다. 이것이 크론이 늙어서 "할아버지"가 되는 이유입니다.

---

## 4. 시간의 법칙

### 4.1 기본 법칙
1. **흐름의 법칙**: 시간은 멈출 수 없으며, 속도만 조절 가능
2. **대가의 법칙**: 시간 조작에는 반드시 대가가 필요
3. **루프의 법칙**: 닫힌 인과는 깨지지 않음

### 4.2 금기
- **시간 정지**: 시도 시 해당 공간이 공허로 전환
- **인과 역전**: 결과를 원인보다 앞에 두면 세계선 붕괴
- **자기 모순**: 과거의 자신을 죽이면 패러독스 발생

---

## 5. 기어 사이의 세계 (던전)

### 5.1 개념
두 마스터 기어 사이에는 무수한 **작은 톱니바퀴들**이 존재합니다. 각 톱니바퀴는 하나의 "시간 영역"을 담당합니다.

### 5.2 특성
| 영역 | 기어 크기 | 시간 속도 | 환경 |
|------|-----------|-----------|------|
| **쇠락의 정원** | 초대형 | 1초 = 100년 | 모든 것이 부식/풍화 |
| **영원의 순간** | 극소형 | 100년 = 1초 | 모든 것이 정지에 가까움 |
| **불안정 지대** | 가변 | 불규칙 | 시간이 무작위로 변화 |

### 5.3 미궁의 역할
크론이 할아버지(미래의 자신)를 구하기 위해 **부품(기어 파편)**을 모아야 합니다. 이 파편은 기어 사이의 세계 곳곳에 흩어져 있습니다.

---

## 6. 시간 감응자

### 6.1 정의
극소수의 인간은 **시간의 흐름을 감지**할 수 있습니다. 이들은 다락방의 존재를 희미하게 인지하거나, 시간 이상 현상에 민감합니다.

### 6.2 해당 인물
- **모라**: 어린 시절 크론의 시간 개입으로 "각성", 시간의 진실을 봄
- **벨로**: 타고난 감응자, 시간 입자 연구에 몰두
- **레이**: 감응 능력 없음 (대조군)

---

## 핵심 요약

> [!IMPORTANT]
> - 우주 = 거대한 시계, 다락방 = 제어실
> - 톱니바퀴 크기 = 시간 속도
> - 회중시계 = 마스터 키 (수명을 연료로 사용)
> - 공허 = 시간이 정의되지 않은 위험 공간

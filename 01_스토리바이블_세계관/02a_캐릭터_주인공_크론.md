# 주인공 - 크론 (CRON)

> *"시간의 수호자, 그리고 파괴자"*

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| **이름** | 크론 (Cron) |
| **나이** | 10세 (시작) → 가변 (시간 여행) |
| **역할** | 플레이어블 주인공 |
| **상징** | 회중시계, 시계바늘 레이피어 |

---

## 외형 변화

```
[어린 크론]          [청년 크론]          [노인 크론]
   ○                   ○                   ○
  /|\   ─(15년)→     /|\   ─(수십년)→    /|\
  / \                 / \                 / \
갈색 머리          흐트러진 머리        백발, 주름
호기심 가득        결의에 찬 눈         지친 채 고독
```

---

## 성격 아크 (노화 단계별)

| 노화 | 성격 | 핵심 감정 | 표현 방식 |
|------|------|-----------|----------|
| **어린** | 호기심 많은 아이, 규칙을 따름 | 공포, 혼란, 탐구심 | 질문이 많음, 두려워하면서도 앞으로 나감 |
| **청년** | 진실을 찾으려는 탐구자 | 희망, 의심, 죄책감 시작 | 대사 톤 무거워짐, "미안해" 증가 |
| **중년** | 자책하는 죄인 | 깊은 자책, 피로 | 한숨, 독백 빈도 증가, 눈빛 흐려짐 |
| **노인** | 희생을 선택하는 수호자 | 체념, 평화 | 짧고 담담한 대사, 미소 |

---

## 죄책감 시작 장면 (2장 중반)

```
레이: "...난 항상 누군가 도와주길 기다렸어."
크론: (내면) "...내가 도와줬을 때, 레이는 웃고 있었는데."

벨로: "실패? 나는 실패해본 적이 없어."
크론: (내면) "벨로도... 내가 실패를 막아줬을 때..."

크론: "잠깐. 내가 도와준 게... 친구들을 망친 거야?"
크론: (주저앉음) "...다 내 탓이야."
```

---

## 핵심 갈등

- **외적**: 할아버지(미래의 자신)를 죽인 범인 추적
- **내적**: 친구들을 구하려던 선의가 그들을 망쳤다는 자책

---

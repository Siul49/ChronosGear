크론의 인생은 단일 직선이 아닌 **자기 참조적 루프**입니다.

```
        ┌──────── 시작점 ────────┐
        │                        │
        ▼                        │
   [어린 크론]                   │
   다락방에서 총성을 듣다        │
        │                        │
        ▼                        │
   회중시계를 줍다               │
        │                        │
        ▼                        │
   시간 여행 시작                │
        │                        │
   (수십 년의 여정)              │
        │                        │
        ▼                        │
   [늙은 크론]                   │
   다락방에서 쓰러지다           │
        │                        │
        ▼                        │
   회중시계를 떨어뜨리다 ────────┘
                ↑
           [패러독스]
    시계는 언제 만들어졌는가?
```

## 🆕 "존재하지 않는 총성" - 할아버지 죽음의 진실

> [!CAUTION]
> **스포일러**: 이 섹션은 4장에서 공개되는 핵심 반전입니다.
> 플레이어는 1~3장 동안 "누가 쐈는가"를 추적하지만, 진실은 완전히 다릅니다.

### 총성의 정체

```
[플레이어의 추측]              [진실]
─────────────────────────      ─────────────────────────
  누군가가 총을 쐈다            아무도 쏘지 않았다
         │                              │
         ▼                              ▼
  총알이 할아버지를 맞혔다       "인과의 비명"이 발생했다
         │                              │
         ▼                              ▼
  범인을 찾아야 한다            시간 균열이 몸을 찢었다
                                        │
                                        ▼
                               "범인"은 시간을 개입한 크론 자신
```

### 인과의 비명 (Causality Scream)

**정의**: 시간 개입이 인과율에 미치는 스트레스가 임계점을 넘을 때 발생하는 현상.

| 특성 | 설명 |
|------|------|
| **소리** | 총성과 유사한 폭발음 (실제 총알 없음) |
| **원인** | 크론의 수십 년간 시간 개입이 한 점에 수렴 |
| **효과** | "그 날"의 다락방에서 시공간 균열 발생 |
| **피해** | 균열에 가장 가까운 존재(크론=할아버지)가 찢김 |

### 할아버지의 죽음 메커니즘

```
[수십 년간의 시간 개입]
         ↓
[모든 개입이 "그 날 밤"에 인과적으로 수렴]
         ↓
[인과율 임계점 돌파]
         ↓
[인과의 비명 발생 = "총성"]
         ↓
[시공간 균열 = 미세한 찢어짐]
         ↓
[늙은 크론 = 할아버지가 균열에 접촉]
         ↓
[존재가 시간 속에서 "분해"]
         ↓
[외부에서 보면 "총 맞은 것처럼" 쓰러짐]
```

### 왜 모두가 용의자처럼 보였는가

| 캐릭터 | 의혹 | 진실 |
|--------|------|------|
| **레이** | 권총 소지, 그날 밤 외출 | 멀리서 총성(비명) 듣고 달려옴 |
| **벨로** | 음파 실험, 폭발 잔해 | 인과의 비명 주파수를 감지함 |
| **모라** | 총기 허가, 알리바이 모호 | 미래에서 이 순간을 보고 왔음 |
| **티카** | 가게 조수, 현장 최근접 | 총알이 없는 걸 제일 먼저 발견 |

### 단서 배치 (챕터별)

| 챕터 | 단서 | 플레이어 해석 | 숨겨진 진실 |
|------|------|---------------|-------------|
| 1장 | 탄피 없음, 총알 자국 없음 | "증거 인멸?" | 총알 자체가 없었음 |
| 2장 | 상처 형태가 이상함 | "특수 무기?" | 균열에 인한 찢어짐 |
| 3장 | 모든 용의자가 알리바이 모순 | "공모?" | 모두 진실을 말함 |
| 4장 | 크론이 과거 개입 기록 발견 | 충격 | 자신이 "원인" |

### 크론의 깨달음 (4장 클라이맥스)

```
크론: "...총알이 없어?"
벨로: "처음부터 없었어. 나는 음파를 분석했지, 총기 폭발음이 아니었어."
크론: "그럼 할아버지는 왜...?"
모라: "시간이야, 크론. 네가 계속 개입한 시간이 한 점에 모였어."
모라: "그 압력이 터진 거야. 인과의 비명."
크론: "...내가? 내가 할아버지를 죽인 거야?"
모라: "그래. 그리고 앞으로도 계속 그럴 거야. 멈추지 않으면."
```

## 나비효과 타임라인

크론이 과거에 개입할 때마다 세계선이 갈라집니다.

### 레이 관련
```
[원래 타임라인]
레이가 불량배에게 맞음 → 스스로 이겨냄 → 자신감 있는 형사

[개입 후 타임라인]
크론이 불량배를 처리 → 레이가 아무것도 안 함 → 타인 의존 → 부패한 형사
```

### 벨로 관련
```
[원래 타임라인]
벨로의 실험 실패 → 좌절 후 반성 → 신중한 과학자

[개입 후 타임라인]
크론이 실패를 되돌림 → 성공만 경험 → 오만 → 미친 과학자
```

### 모라 관련
```
[원래 타임라인]
모라 부모님 사고 → 슬픔을 딛고 성장 → 정의로운 검사

[개입 후 타임라인]
크론이 사고를 막으려 시간 왜곡 → 모라가 공허를 목격 → 시간 정지 신봉자
```

## 회중시계의 순환

회중시계는 **인과적으로 폐쇄된 루프** 안에 존재합니다.

```
1. 어린 크론이 회중시계를 줍는다
         ↓
2. 수십 년간 사용하며 노화
         ↓
3. 늙은 크론이 "그 날"에 쓰러진다
         ↓
4. 회중시계가 떨어진다
         ↓
5. 어린 크론이 주운다 ← (1로 돌아감)
```

### 존재론적 질문
- 시계는 언제 만들어졌는가? → **해답 없음**
- 시계의 창조자는 존재하는가? → **불명** (전설에서만 언급)
- 루프를 끊으면 시계는 사라지는가? → **진 엔딩의 핵심**

## 핵심 요약

> [!IMPORTANT]
> - 크론의 시간과 세계의 시간은 **완전히 다르게 흐름**
> - 선의의 개입이 친구들을 **타락시킴** (나비효과)
> - 회중시계는 **시작도 끝도 없는** 패러독스적 존재
> - 모든 타임라인은 결국 **"그 날"의 다락방**으로 수렴

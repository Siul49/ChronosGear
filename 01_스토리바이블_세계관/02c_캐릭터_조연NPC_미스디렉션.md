# 조연 NPC - 추리 미스디렉션용

> 플레이어가 "누가 쐈는가"를 추적할 때 **미스디렉션**을 제공하는 캐릭터들입니다.
> 모두 **무고**하지만, 각자 총성과 연결되는 의혹을 가집니다.

---

## 1. 티카 (TICA) - 현장 최초 발견자

> 시계 가게의 조수, 총알 부재의 최초 증인

### 기본 정보

| 항목 | 내용 |
|------|------|
| **이름** | 티카 (Tica) |
| **나이** | 18세 (사건 당시) → 33세 (현재) |
| **역할** | 조연 / 정보원 |
| **직업** | 시계 가게 조수 → 골동품상 운영 |
| **상징** | 톱니바퀴 귀걸이, 손목의 화상 자국 |

### 의혹 요소

| 의혹 | 설명 |
|------|------|
| 🔴 총기 소지 면허 | 가게 경비 목적으로 권총 보유 |
| 🔴 그날 밤 근무 | 사건 시각에 가게 1층에 있었음 |
| 🔴 할아버지와 다툼 | 임금 문제로 말다툼 목격됨 |

### 진실

- 총성을 듣고 달려갔을 때 **총알이 없는 것**을 제일 먼저 발견
- 할아버지가 "시간이 분해된다"고 중얼거리는 것을 들음
- 무고하며, 후반에 중요 정보(총알 부재, 마지막 말) 제공

### 대사 예시

> *"총소리는 분명히 들었어요. 근데... 총알이 없었어요. 아무리 찾아도."*
> *"할아버지가 마지막에 뭐라고 했냐고요? '내가... 내가 했어'라고 했어요."*

## 2. 오리 (ORI) - 사인 조작 의혹자

> 마을 의사, 상처 형태 이상의 유일한 증인

### 기본 정보

| 항목 | 내용 |
|------|------|
| **이름** | 오리 (Ori) |
| **나이** | 45세 (사건 당시) → 60세 (현재) |
| **역할** | 조연 / 증인 |
| **직업** | 마을 의사 → 은퇴 |
| **상징** | 낡은 청진기, 떨리는 손 |

### 의혹 요소

| 의혹 | 설명 |
|------|------|
| 🔴 사인 조작 | 사망 진단서에 "총상"이라고 기재 |
| 🔴 뇌물 수수 소문 | 진단서 조작으로 돈 받았다는 소문 |
| 🔴 할아버지와 비밀 | 정기적으로 할아버지를 비밀 진료 |

### 진실

- 할아버지의 상처가 **총상이 아님**을 알았지만, 설명할 방법이 없어서 "총상"이라 씀
- 실제 상처는 "몸이 안에서 찢어진 것 같은" 형태
- 무고하며, 2장에서 상처 형태의 이상함을 증언

### 대사 예시

> *"40년 의사 생활 동안 저런 상처는 처음 봤어. 총알이 들어간 게 아니라... 안에서 바깥으로 터진 것 같았지."*
> *"총상이라고 쓴 건 거짓말이야. 하지만 뭐라고 써야 했겠어? 시간에 찢겼다고?"*

## 3. 그레이 (GRAY) - 과학적 증거 제공자

> 벨로 연구팀원, "인과의 비명" 음파 분석가

### 기본 정보

| 항목 | 내용 |
|------|------|
| **이름** | 그레이 (Gray) |
| **나이** | 28세 (현재) |
| **역할** | 조연 / 기술 단서 |
| **직업** | 벨로 연구소 수석 연구원 |
| **상징** | 보호 고글, 방사선 배지 |

### 의혹 요소

| 의혹 | 설명 |
|------|------|
| 🔴 음파 무기 개발 | 총성과 유사한 주파수 발생 장치 연구 |
| 🔴 실험 사고 기록 | 연구소에서 "폭발음" 사고 다수 |
| 🔴 할아버지 방문 기록 | 사건 1주일 전 시계 가게 방문 |

### 진실

- "인과의 비명" 주파수를 **사후에** 분석한 최초의 인물
- 총성이 아닌 "시간 입자 붕괴음"임을 과학적으로 증명
- 무고하며, 3장에서 핵심 과학 증거 제공

### 대사 예시

> *"이 음파 패턴은 화약 폭발이 아니야. 주파수가 완전히 달라. 마치... 무언가가 찢어지는 소리 같아."*
> *"벨로 박사님은 이걸 '인과의 비명'이라고 불러. 시간 개입이 한계에 도달하면 발생하는 현상이래."*

## 총성 단서 배치 - 추리 시스템 통합표

> 모든 캐릭터가 "범인"처럼 보이게 하는 단서 배치

| 캐릭터 | 총성 연결 단서 | 플레이어 오해 | 진실 |
|--------|---------------|---------------|------|
| **레이** | 권총 소지, 그날 밤 외출 | "형사니까 총 쐈겠지" | 총성 듣고 달려옴 |
| **모라** | 총기 허가, 알리바이 불명 | "미래에서 왔다며?" | 미래에서 목격하러 옴 |
| **벨로** | 폭발 실험, 음파 장비 | "과학자니까 무기 만들었겠지" | 사후 분석만 함 |
| **티카** | 권총 보유, 현장 최근접 | "다툼 후 화가 났겠지" | 총알 부재 발견자 |
| **오리** | 사인 조작, 비밀 진료 | "증거 인멸 공범?" | 설명 못해서 거짓 기재 |
| **그레이** | 음파 무기, 가게 방문 | "실험 대상이었나?" | 주파수 분석가 |

> [!IMPORTANT]
> **6명 모두 의혹**을 갖지만, **6명 모두 무고**합니다.
> 진짜 "범인"은 존재하지 않으며, 원인은 시간 개입 자체입니다.

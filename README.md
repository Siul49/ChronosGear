## 🎮 게임 개요

| 항목 | 내용 |
|------|------|
| **장르** | 2D 쿼터뷰 액션 어드벤처 |
| **시점** | Isometric (점프 없음) |
| **아트** | 스팀펑크 + 코즈믹 호러 도트 |
| **테마** | 시간, 운명, 희생 |

## 🔑 핵심 키워드

- **시간 루프**: 할아버지 = 미래의 자신
- **톱니바퀴**: 크기가 시간 속도를 결정
- **선의의 비극**: 친구를 돕다가 파멸을 불러옴
- **철학적 딜레마**: 시간을 흘릴 것인가, 멈출 것인가
